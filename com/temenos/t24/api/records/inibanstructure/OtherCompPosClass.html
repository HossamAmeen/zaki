        public TField getOtherCompLen()
        Returns the OtherCompLen

        Field description: Associated Multi-value field with OTHER.COMP.POS Field to denote the length of the OTHER.COMP.POS component in the IBAN For eg: If the value of IB.COUNTRY.CDE.POS is 1 and its corresponding length is 2, and if the value of IB.CHK.DIGITS.POS is 5 and its corresponding length is 3, Then system considers the missed out position (which is from 3, 2 positions) as control characters and populates the OTHER.COMP.POS as 3 and OTHER.COMP.LEN as 2.
