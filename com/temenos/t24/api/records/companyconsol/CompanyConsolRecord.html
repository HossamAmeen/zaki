        public void addComConsolFrom(TField param)
        Appends the specified ComConsolFrom to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Company code to identify the companies whose details are to be included. In this field are listed the Companies whose financial details are to be included in a combined report. The Companies specified can only be the operative companies. It is not possible to specify a consolidation company in this field. For each group all the base operational companies to be included in that group must be specified. Validation Rules: The 9-digit company code or the 3-digit company mnemonic. This is a multivalue field. Mandatory input. Must be a valid Company on the Company file. Must be an operative (non consolidation) Company. Company code can not be duplicated within this field.
