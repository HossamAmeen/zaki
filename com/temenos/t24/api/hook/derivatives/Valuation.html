      • calculateCreditExposureValues

        public CreditExposure calculateCreditExposureValues(String applicationName, TBoolean isRunningInCob, CreditExposureParameters creditExposureParameters, DxTransactionRecord derivativeTransactionRecord)
        This interface enables the implementer to calculate the credit exposure values, based on the add-on rates.
        This interface is invoked while running the enquiry SC.VAL.COST.
        The derivatives transaction record, lots, trade type, call option, buy/sell and internal strike price are passed in to calculate the credit exposure values.

        T24 Details: The EB.API hook used by this interface is DX.PARAMETER.CR.EXP.CALC.API.HOOK. The T24 field specifying this hook is the CR.EXP.CALC.API field in DX.PARAMETER.
        If an exception is thrown in the implementing class, the enquiry will be cancelled; if running within a service the error will additionally be stored to the EXCEPTION.LOG.
        Parameters:
        applicationName - Name of the application from where this interface is being triggered
        isRunningInCob - Denotes whether the calculation is invoked from close of business processing
        creditExposureParameters - Denotes the parameters required for calculating credit exposure such as lots, tradetype, callput, buysell, internal strike price and application product
        derivativeTransactionRecord - Data about the derivatives transaction such as dx transaction, lots etc
        Returns:
        CreditExposure - calculated credit exposure data such as replacement credit exposure, internal credit exposure etc

      • calculateNetCost

        public void calculateNetCost(String applicationName, NetCostParameters netCostParameters, CurrencyAmount taxAmount, TNumber netCost)
        This interface enables the implementer to calculate the net cost of a derivatives trade or order in trade currency.
        This interface is invoked when validating a DX.TRADE or DX.ORDER.
        The derivatives contract code, maturity date, lots, call option, buy/sell, currency market, price, tax amount are passed in to calculate the net to the customer.

        T24 Details: The EB.API hook used by this interface is DX.PARAMETER.COST.CALC.API.HOOK. The T24 field specifying this hook is the COST.CALC.API field in DX.PARAMETER.
        If an exception is thrown in the implementing class, the transaction will be cancelled.
        Parameters:
        applicationName - Name of the application from where this interface is being triggered
        netCostParameters - Denotes the parameters required for calculating the net cost such as contract code, maturity date, lots, call option, buy/sell, currency market, price and tax amount
        taxAmount - The tax data containing the total tax amount and its currency
        netCost - The net cost in trade currency
        Returns:
        nothing

      • getRevaluationMarginDetails

        public TStructure getRevaluationMarginDetails(String applicationName, TBoolean isRunningInCob, String marginDetailTableName, RevaluationRequest revaluationRequest, RevaluationDetail revaluationDetail)
        This interface enables the implementer to re-calculate the margin values during revaluation of derivative trades.
        This interface is invoked when authorising a DX.REVALUE record for the specified positions and by DX.COB.REVALUE during COB processing to re-calculate the margin for all customer positions.
        This interface is also invoked when updating the accounting entries for a transaction during DX.TRADE or DX.CLOSEOUT.
        The derivatives revaluation setup record and process record are passed in to re-calculate the values for the initial or variation margin types.

        T24 Details: The EB.API hook used by this interface is DX.MARGIN.CALC.PROGRAM.NAME.HOOK. The T24 field specifying this hook is the PROGRAM.NAME field in DX.MARGIN.CALC
        If an exception is thrown in the implementing class, the transaction will be cancelled; if running within a service the error will additionally be stored to the EXCEPTION.LOG.
        Parameters:
        applicationName - Name of the application from where this interface is being triggered
        isRunningInCob - Denotes for running in close of business or not
        marginDetailTableName - Name of the revaluation detail table. This always starts with DX.REVAL.DET.**** and is suffixed with the id of the DX.MARGIN.CALC record
        revaluationRequest - This is data about the derivatives setup such as customer portfolio, dealer desk, price set etc
        revaluationDetail - This is data about the derivatives process such as derivative position, transaction id etc
        Returns:
        TStructure - a revaluation detail record for the table defined in the marginDetailTableName parameter.
