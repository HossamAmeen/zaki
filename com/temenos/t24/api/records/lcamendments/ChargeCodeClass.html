      • getChargeStatus

        public TField getChargeStatus()
        Returns the ChargeStatus

        Field description: CHARGE.STATUS Code to identify the required status of the Commission/Charge, whether it is to be collected immediately or deferred or waived. When the status is NULL, the Commission/Charge amount will be calculated and written in LC.ACCOUNT.BALANCES. No accounting will take place. This has to be collected later using Operation C in LC, at which time Tax if any will also be collected. If the status is 2, Commission/Charge will be collected immediately along with Tax if any. If the status is 15, the charge is waived and cannot be collected in the future. If is stored in LC.ACCOUNT.BALANCES as a charge that has been waived. Validation Rules: Valid input NULL or 2.

        Parameters:
        Returns:
        TField - The ChargeStatus
