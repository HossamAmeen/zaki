com.temenos.t24.api.system

Class Session

Packaged in : EB.SessionApi.jar

Provides APIs for accessing data and variables relating to the current session of the user or process.
Many methods are simple getters for some of the important COMMON variables in T24, like the current user or company record.

      • getOnlineStatus

        public String getOnlineStatus()
        Returns a value indicating whether the system is running end of day processing.

        T24 Details: : This returns a value based on the OP.MODE field in SPF. The possible values are
        • 'ONLINE' - normal operation, users may INPUT transactions.
        • 'BATCH' - end of day processing. Most applications will not allow input during the close of business processing, but a few (eg FUNDS.TRANSFER) can.
        • 'RECOVERY' - database restoration.
        Parameters:
        Returns:
        String - Returns a value indicating whether the system is running end of day processing.
        Throws:
        T24CoreException if there is any error returning the online status.
