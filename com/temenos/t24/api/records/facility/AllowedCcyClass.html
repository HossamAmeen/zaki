      • getAllowedCcy

        public TField getAllowedCcy()
        Returns the AllowedCcy

        Field description: Field to indicate list of currencies in which loans could be drawn under the tranche. When a facility is granted, there could be stipulations that permit drawings in specific currencies only. This field is used to list out the currencies in which loans could be drawn (Deal.Currency). If loans are allowed in certain currencies those currencies could be defined by multi-valuing this field. If loans are allowed except in one or more currencies, such exceptions could be listed in Restrict.Ccy When loans are rolled over, this validation is checked again. Validation Rules: 3 alpha currency mnemonic or 1-3 numeric character currency code Multi-value field to define more than one currency in which drawings are allowed. Input allowed only if Restrict.Ccy is not defined Optional input. If no input is made drawings are allowed only in Sl.Currency

      • getHoldingRate

        public TField getHoldingRate()
        Returns the HoldingRate

        Field description: Defines that rate to be defaulted to the loan (if drawn in the respective allowed currency) in field Holding Rate When loan is drawn, there could be circumstances when the interest rate is not determinable in advance. The interest rate is agreed/fixed at a date prior to the maturity of the loan, but well after the value date. In such cases, the bank may accrue interest at a holding rate for the interim period. Once the actual interest rate is defined in the loan contract, the accruals done at the holding rate is dropped and are made at the newly defined interest rate. Validation Rules: 1-10 numeric characters plus decimal point Must be greater than zero Maximum value allowed is 100
