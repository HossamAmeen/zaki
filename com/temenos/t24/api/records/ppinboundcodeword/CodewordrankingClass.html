        public TField getCodewordpriorityforpd()
        Returns the Codewordpriorityforpd

        Field description: Specifies the codeword priority for product determination process. The codeword with the highest priority will be considered as the 'Final Product Codeword' for the payment order. If there are multiple codewords with the same codeword priority for PD, then the 1st record is selected and the codeword corresponding to that record is set as Final Product Codeword. Possible values: Value ranges from 0 to 9.

        Field description: This field is multivalue which specifies the order (sequence) of the record in the application.
