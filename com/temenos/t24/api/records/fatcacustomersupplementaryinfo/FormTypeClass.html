      • getCutOffDate

        public TField getCutOffDate()
        Returns the CutOffDate

        Field description:

        This is the date by when the document specified in FORM.TYPE field has to be received. This will be defaulted based on the settings in FATCA PARAMETER.

        For a new client, this will be either the Opening or Request Date plus the number of days set in NEW CLNT DOC DAYS.

        For an existing client, the cut-off date will be determined based on the number of days specified in parameter (EXIST.FFI.DOC.DAYS or OTH.EXIST.DOC.DAYS).

        The cut-off date can be manually changed. If the documents are not received within the cut-off date above, the account status will be changed to recalcitrant, provided automatic classification facility is available.

        The field is part of the associated multi-value set starting with FORM.OWNER.

        Validation rules

        1-9 Type D Standard date format -(YYYYMMDD)

        Parameters:
        Returns:
