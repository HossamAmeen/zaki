        public ChargeAmount calculateChargeAmount(TNumber customerId, Amount chargeBaseAmount, TNumber currencyMarket, String customerRate, String chargeCurrency, String currencyType, TransactionContext transactionContext, List<CalculationData> calculationData)
        This interface enables the implementer to calculate the charge amount to be applied against a Letter of Credit.
        This interface is invoked when providing the charge details for a Letter of Credit.

        T24 Details: The EB.API hook used by this interface is LC.PARAMETERS.CHARGE.RTN.HOOK.
        The T24 field specifying this hook is the CHARGE.ROUTINE field in LC.PARAMETERS table. This interface is invoked during validation of LETTER.OF.CREDIT record.
        If an exception is thrown in the implementing class, an error will be raised and the Letter of Credit record will not be committed. If the error occurs within a service it will also be logged.
        Parameters:
        customerId - Applicant, Beneficiary or Thirdparty ID based on the party charged
        chargeBaseAmount - portion of the Letter of Credit amount for which the charges are to be applied. t24 Details: This data comes from the CHARGE.BASE.AMOUNT field and COMMISSION.CURRENCY field in ter.OF.CREDIT table.
        currencyMarket - Currency Market
        customerRate - exchange rate to be used between the charge debit account and the charge currency
        chargeCurrency - Currency of charge code in FT.COMMISSION.TYPE
        currencyType - Currency Type. The list of possible values is: charge account currency or LetterOfcredit currency
        transactionContext - Data required for transaction
        calculationData - Input values for charge amount calculation
        Returns:
        ChargeAmount - s the charge amount.
