        public TField getInwardBulkIds()
        Returns the InwardBulkIds

        Field description: Identifies which Inward Bulk Standing Order is to be executed. It must be a valid ID in the Live Inward Bulk Standing Orders file. The contents of this field must correspond to a valid entry on the Inward Bulk Standing Order file, this being Field 0 ACCOUNT.NO.DATE. Validation Rules: Up to 24 type A (alphanumeric) characters as follows: - The first part of the ID identifies the Account Number for which the Inward Bulk Standing Order has been defined. - The second part of the ID is separated from the Account Number by a '-' and will be a date. Associated multivalue field. (Mandatory input.)
