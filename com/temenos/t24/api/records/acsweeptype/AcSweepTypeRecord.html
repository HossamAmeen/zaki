        Appends the specified SweepTrigger to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: This field allows to define transaction type. It will only be allowed for Maintenance sweep type.This type of record will only be allowed to be used by AC.ACCOUNT.LINK application and not allowed for AC.CASH.POOL This is a multivalue field with 3 options CUST-DR - If this option will be selected, the sweep will be triggred in case the sweeping accounts balanceis less than the minimum amount defined,only if during the current processing day, there was a customer initiatedtransaction posted on the account BANK-DR - If this option will be selected, the sweep will be triggered in case the sweeping accounts balanceis less than the minimum amount defined,only if during the current processing day, there was a bank initiatedtransaction posted on the account. NULL/BLANK If this option will be selected,The sweep will be triggered if the sweeping accounts balance is lessthan the minimum amount defined, irrespective of the transaction types posted on the account. If both CUST-DR and BANK-DR options are selected, the system will trigger the sweep only if a customer or a bankinitiated debit transaction were posted on the sweeping account, otherwise the sweep execution will be skipped untilthe next scheduled date.
