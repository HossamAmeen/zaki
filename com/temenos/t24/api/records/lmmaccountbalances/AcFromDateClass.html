        public TField getAcAccrAmt()
        Returns the AcAccrAmt

        Field description: The set of fields from AC.FROM.DATE to AC.ACT.ACCR form an association containing interest accrual data. A new multivalue is created each time a month end is passed, or the principal or interest rate is changed. AC.ACCR.AMT contains the amount of interest accrued for the period rounded to the number of decimals for the currency. The other 6 fields are for date accrued to (AC.TO.DATE), number of days in the accrual period (AC.NO.DAYS), interest rate (AC.RATE), rounded accrued amount (AC.ACCR.AMT), and unrounded accrued amount (AC.ACT.ACCR). These 7 fields store data in reverse chronological order (the most recent being found in the first multivalue). Validation Rules: Standard T24 amount field of up to 19 characters.
