        public ForexPosition getForexPosition(String principalCurrency, GapClass gapClass, PositionContext positionContext)
        This interface is used to obtain the position class values and calculates interest rate and obtains notional currency.
        This interface is invoked in online mode

        T24 Details:The EB.API hook used for this interface is PM.PARAMETER.FX.SYNTH.RTN.HOOK.
        The T24 field specifying this hook is the PM.FX.SYNTH.RTN field in PM.PARAMETER .
        Updates the position class with new values.
        Parameters:
        principalCurrency - The currency on which to base the calculation,this is obtained from the forex application
        gapClass - The name of the position class obtained from FX application is obtained through common variables
        positionContext - The values required during a transaction.
        Returns:
        ForexPosition - the position class rate, amount and currency.

      • getForwardRateAgreementPosition

        public ForwardRateAgreementPosition getForwardRateAgreementPosition(PositionContext positionContext, FraDealRecord fraDealRecord)
        This interface is used to append the core position management from the FRA.DEAL application.
        For FRA HEDGE purchase deal the position classes FRXDS, FRXDM should hold the reference price instead of the FRA Contract price. For a FRA HEDGE Sale Contract , the position classes FRXLS, FRXLM should hold the reference price instead of the FRA contract price This interface is invoked in online mode

        T24 Details:The EB.API hook used for this interface is PM.PARAMETER.FRALONGRATE.HOOK.
        The T24 field specifying this hook is the FRA.LONG.RATE field in PM.PARAMETER .
        Updates the position class with new values.
        Parameters:
        positionContext - The values required during a transaction.
        fraDealRecord - The record of the SC.TRADING.POSITION file appropriate to the securityId parameter
        Returns:
        ForwardRateAgreementPosition - the position class and processed.

      • getSecurityPosition

        public SecurityPosition getSecurityPosition(String className, ScTradingPositionRecord scTradingPositionRecord, PositionContext positionContext)
        This interface makes the PM.GAP enquiry to project constant YTM and varies the bond nominal on every day cob such that the premium/discount will be amortised evenly over the bond period.
        This interface is invoked during COB when the amortisation of bond or discount is required.

        T24 Details:The EB.API hook used for this interface is PM.SC.PARAM.YEILDTOMATURITY.HOOK.
        The T24 field specifying this hook is the BOND.YTM.GAP field in PM.SC.PARAM .
        Updates the position class with new values.
        Parameters:
        className - the name of the position class obtained from SEC.TRADE/FX application
        scTradingPositionRecord - The record of the SC.TRADING.POSITION file appropriate to the securityId parameter
        positionContext - The values required during a transaction.
        Returns:
        SecurityPosition - true if the position was calculated, false otherwise.
