      • getReplaceChars

        public TField getReplaceChars()
        Returns the ReplaceChars

        Field description: This field is used to specify that certain characters or text should be replaced with the characters specified in REPLACE.WITH when records are saved from T24 files into the data library. These fields will be checked when the "V"erify mode is taken in DL.DEFINE to save records into a unit in the data library (OPERATION in DL.DEFINE is set to "S"). For example: You have spent some considerable time creating PD advices, which you feel may be of benefit to other T24 clients and you therefore want to share these records with everyone. However, these advices contain the name of your bank, which you do not want other T24 clients either seeing this information or using it inadvertently. Therefore, you can specify in REPLACE.CHARS which characters/text you want removed. e.g. REPLACE.CHARS T24 Test Bank REPLACE.WITH ***Bank Name *** REPLACE.CHARS 183535902 REPLACE.WITH *Vat.No** REPLACE.CHARS must be replaced by an equal number of characters in REPLACE.WITH. Validation Rules: Optional input Multi-value field Fields REPLACE.CHARS to REPLACE.WITH form a multi-value associated set. 1-50 Type "A" (alphanumeric) characters

        Parameters:
        Returns:
