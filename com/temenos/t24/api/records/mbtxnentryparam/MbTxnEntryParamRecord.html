        public void addExcAsset(java.lang.CharSequence param)
        Appends the specified ExcAsset to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: This is a multivalue field used to define the asset types that should be excluded from being displayed in the enquiry TXN.ENTRY.MB. For example : For a contract AA1, there is a SPEC entry with asset type S1. If S1 asset type is entered in the field EXC.ASSET, then when the enquiry TXN.ENTRY.MB is run for the contract AA1, this SPEC entry with asset type S1 is not displayed in the ouput of the enquiry.”
