      • addLocalRoutine

        public void addLocalRoutine(TField param)
        Appends the specified LocalRoutine to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: By using local routines regional developers can update the table and return spreads and/or interest rates based upon decisions in their own routines. This field allows local routines to be called when this record is accessed by the TERMRATE routine. Two parameters are passed to the jBC routine, and returned by, the local routine: - INTEREST.RATE the derived Periodic interest rate - MARGIN which can be passed from the calling process In JAVA hook, the new rates and the modified records will be returned through the 4th, 5th and 6th parameter If more than one local routine is defined then each routine will be called in turn with the above parameters. As an example, Money Market Auto Rollovers can use a routine to determine new rates and/or spreads based on locally developed spread tables related to department and customer groups. See Helptext for Money Market ROLLOVER.INT.RATE.

        Validation Rules: Optional input. Multi Valued field up to 35 alphanumeric characters in length. should have a value in EB.API file of TYPE as 'BASIC' if the routine is JBase routine or TYPE as 'METHOD' which implements an interface defined in the EB.API record HOOK.PI.LOCAL.RTN for Java API.

        Parameters:
        param - the LocalRoutine to append
        Returns:
        nothing

      • getIntTolerance

        public TField getIntTolerance()
        Returns the IntTolerance

        Field description: Interest percentage tolerance allowed on money market contracts. A tolerance entered here will be used to check if an over-ride is required for any interest rate difference found on any money market contracts entered. The existence of a default INT.TOLERANCE in LMM.INSTALL.CONDS indicates that a check for an interest rate difference over-ride should be made on money market contracts. If the difference between the rate entered and that calculated exceeds the percentage tolerance entered here then an over-ride is required.

        4 Numeric character input up to 100% (Optional input) This field may only be entered for records in this file with a leading sequence number of '01'.

        Parameters:
        Returns:
        TField - The IntTolerance
