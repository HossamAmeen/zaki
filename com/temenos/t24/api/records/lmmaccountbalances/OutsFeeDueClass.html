      • getCumFeeRec

        public TField getCumFeeRec()
        Returns the CumFeeRec

        Field description: The amount of the fee which has been received. This is the amount of the particular fee which has been received. The field is updated by the receipt entry system in respect of the payment of fees. This field is signed and is always positive. Associated Field Considerations. Each fee defined on the schedules will have a set of multivalues entered into this association on the due date if it has not been successfully liquidated. This may be because the contract is set to manual liquidation or because certain account posting restrictions exist. The multivalue association OUTS FEE DUE to LAST DATE REC contains details off charges on the contract. The associated fields are as follows:- OUTS FEE DUE FEE DUE DATE FEE CHRG CODE CUM FEE REC LAST DATE REC Validation Rules: Upto 14 AMT (Standard amount format) type characters plus decimal point. Multivalue association: OUTS FEE DUE to LAST DATE REC. (For LD.LOANS.AND.DEPOSITS only.)
