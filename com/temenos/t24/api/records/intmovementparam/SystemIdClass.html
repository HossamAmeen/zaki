        public TField getGrossNetInt()
        Returns the GrossNetInt

        Field description: Flag to determine if Interest if paid Gross or Net of Tax for the associated T24 Application. This flag indicates if Interest is paid to the customer gross or net of tax. It is to be noted that applications like MM, LD etc are hard-coded within T24 to post only the Net Interest to the Customer. Validation Rules: Mandatory field and only the values "GROSS" or "NET" are accepted.

        Parameters:
        Returns:
        TField - The GrossNetInt

      • getSystemId

        public TField getSystemId()
        Returns the SystemId

        Field description: Start of a Multi value set defining the extract conditions. The contents in this field will define any valid T24 APPLICATION for which the rest of the Multi Value set applies. This field is Sub-Valued to enable input of more than one Application Id. The value "ALL" is also allowed in which case it refers to ALL T24 Applications. Care must be taken if "ALL" is input since the Transaction codes defined in the following fields will have to be unique across all applications within T24. Valid Applications can be AC, IC1, IC2, MM, LD, etc. Validation Rules: Mandatory field, 2 to 5 characters.

        Parameters:
        Returns:
        TField - The SystemId
