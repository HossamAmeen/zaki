      • getLevel

        public TField getLevel()
        Returns the Level

        Field description:

        Level will be used to group the conditions and evaluate based on the condition level. When no value is specified it will be considered as previous conditions level value.

        An example is provided below:

        To be considered a non-US account, the client has to provide W-8BEN or FGN.PASSPORT. In this case, the Level field will be blank.

        If the requirement is to provide (W-8BEN-E and FIN.REPORT) or (OWNER.CONFIRMATION), then W-8BEN-E and FIN.REPORT can be set as LEVEL 1 (to be evaluated together) and OWNER CONFIRMATION as LEVEL 2 (this will be evaluated after Level 1 has been evaluated and no match has been found).

        Validation rules

        Numeric input

        Must be defined in ascending order.

        Parameters:
        Returns:
        TField - The Level

        public TField getOperand()
