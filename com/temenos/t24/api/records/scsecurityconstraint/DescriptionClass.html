      • getValidityDate

        public TField getValidityDate()
        Returns the ValidityDate

        Field description: Multi-valued field associated with all the fields from NATIONALITY to RESTRICTION. Specifies the Validity Date of the constraint. If the restriction on being the beneficial owner only lasts until a particular date then the final date upon which the restriction applies should be entered in the VALIDITY.DATE field associated with the Nationality or Residence to which this limited restriction applies. If the VALIDITY.DATE field associated with a Nationality is unlimited then the VALIDITY.DATE field should be left blank. It is possible to input a VALIDITY.DATE in the past. Validation Rules: Up to 9 type D date characters, (Standard Date format in range 1950-2049).

        Parameters:
        Returns:
        TField - The ValidityDate
