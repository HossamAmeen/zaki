        Field description: Contains all the Override messages which the user agreed to during input. In the event that the user confirms the Override, the full text of the actual Override Message will be stored, for reference purposes only, in this multivalue filed. For example, if the user inserts a new line type 'DETAIL' and no are entered, then when the line is validated a warning bell will sound and a message will appear, at the foot of the screen, which reads: 'NO DETAILS PRESENT' This will draw the user's attention to the line entered, which may be a mistake and can then be corrected, or, on the other hand, it may be correct input and so the user confirms that his line definition is really intended.
