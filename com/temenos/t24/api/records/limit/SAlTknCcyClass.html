      • getSAlTknAmt

        public TField getSAlTknAmt()
        Returns the SAlTknAmt

        Field description: Identifies, together with the previous field S.AL.TKN.CCY the various sub allocation taken outstanding amounts by currency. The total of these various amounts for all currencies can be seen, expressed in the Limit currency, in the field 28.1 'SUB ALLOC TAKEN'. Note: (i) All amounts which are in a currency different than the LIMIT CURRENCY (Field 1) are converted at the currency table middle rate to generate the 'SUB ALLOC TAKEN' (Field 28.1). In accordance with the frequency defined in the LIMIT.PARAMETER table, these amounts will be revalued with the last currency table middle rate to recalculate a new total SUB ALLOC TAKEN. Any excess of the ONLINE LIMIT amount due to the limit file revaluation will be indicated as such in the OVERDRAFT AND EXCESS report. Validation Rules: 1 to 14 type AMT (standard amount format) characters. System generated. Subvalue field associated with Field 33.1.1. It is also part of the association of fields 25 to 45.
