        public TField getCardIdPosn()
        Returns the CardIdPosn

        Field description: Defines the screen co-ordinates for display of the card id. The card id is preceded by a multilanguage translation of the text "Card id:" and is displayed at the co-ordinates specified on the signature card display. Validation Rules: 1 - 3 numeric ',' 1 -3 numeric Enter as XXX,YYY where XXX = X axis co-ordinate in the range 1-1000 YYY = Y axis co-ordinate in the range 026-750 Co-ordinates are converted at display time into a fraction of the available screen area of the terminal being used for display. The 026 minimum on the Y axis allows for the status line on the signature card.
