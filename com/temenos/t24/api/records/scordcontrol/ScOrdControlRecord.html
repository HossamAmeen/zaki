        public TField getErrorCondition()
        Returns the ErrorCondition

        Field description: No input field that will contain details of any error encountered by the bulk processing phantom. If the phantom encounters an error then the ID of the SEC.TRADE transaction will be left in the PROCESSING.ID field and the error message reported here. The phantom will then close down. The stage of authorisation will be recorded in the SC.ORD.PROCESSING record for the SEC.TRADE concerned. Each stage of bulk process authorisation will give rise to a separate JOURNAL record. If there is anything in this field then the Temenos helpdesk should be contacted immediately. Validation Rules: No input field.

      • getSleepTime

        public TField getSleepTime()
        Returns the SleepTime

        Field description: Field to allow the user to specify a length of time in seconds for the bulk processing phantom to sleep between authorisations. The phantom SC.BULK.PROCESS works by selecting SC.ORD.PROCESSING (which contains the IDs of all the SEC.TRADE transactions which have been selected for bulk processing but have not yet been authorised) and authorises them. It will then sleep for the number of seconds input here before selecting the SC.ORD.PROCESSING file again. Validation Rules: Input of a number between 0 and 999 allowed. Optional field. No default value. Leaving the field blank will result in the phantom process not sleeping between selections of the SC.ORD.PROCESSING file.

        Parameters:
        Returns:
        TField - The SleepTime
