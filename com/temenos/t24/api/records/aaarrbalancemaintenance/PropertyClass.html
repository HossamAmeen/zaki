      • getAdjBankAmt

        public TField getAdjBankAmt()
        Returns the AdjBankAmt

        Field description: This field denotes the bill adjustment for BANK amount for the given property. This can be signed. Sign given here signifies if we are decreasing or increasing the bill property amount.

        Example: A bill for loan product, which has constant payments, is captured with a monthly repayment amount of 10000, in which the principal part is say 3000 and interest part is 7000. Suppose, the interest amount is to be increased by say 500, then a value of -500 has to be given here, which would increase the Interest amount to 7500.

        Validation rules Standard t24 signed amount field

        Returns:

      • getNewSusPropAmt

        public TField getNewSusPropAmt()
        Returns the NewSusPropAmt

        Field description: When a bill is captured or adjusted (taken over from non-T24 system or any past bills that are manually captured for the arrangement) if the arrangement is suspending then it is possible to capture/adjust the suspended amount of interest or charge within the bill separately from the outstaning amount using this field. This field allows to capture the new outstanding suspension amount, if there was any suspension amount for the bill for the property the difference between this field and old suspension amount will be reversed

        Validation Rules

        Either NEW.SUS.PROP.AMT or ADJ.SUS.PROP.AMT can only be captured, not both Arrangement should be in suspended status Effective date of the bill should be great than the suspension date of the arrangement

        Parameters:
        Returns:
