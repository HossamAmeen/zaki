      • getRebalance

        public TField getRebalance()
        Returns the Rebalance

        Field description: Define nature of each cell. Depending of the nature of cell, rebalancing tool have different process to generate proposed orders. Blank : Rebalancing tool doesn't process this cell. MOD : "Model portfolio" Rebalancing tool get directly the security set for this cell. Assign to cell who define exactly the security. RCD : "Recommendation" Rebalancing tool used recommended stock list to proposed orders. Assign to cells who securities with criteria (asset, currency, industry sector ...). TGHT : "Through" Rebalancing tool used linked matrix to rebalance the portfolio. NOT AVAILABLE CCY : Rebalancing tool generate FX spot transactions for each cell defined as CCY. PIVOT : Defined the pivot currency when rebalancing generate FX spot orders. HDG : Rebalancing tool generate FX forward transactions for each cell defined as HDG. Validation Rules: Only MOD, RCD, TGHT can be entered. Cannot be set on a total cell. A total cell cannot be rebalance. (see User Guide) For TGHT option, a linked matrix must be set. For CCY option a PIVOT cell must be set.

        Parameters:
        Returns:
        TField - The Rebalance
