        public TField getTierSourceType()
        Returns the TierSourceType

        Field description: Key to the AA.SOURCE.CALC.TYPE application and used for calculating Tier Base Balance for associated CALC.PROPERTY. For an interest/charge condition, if the calculated Tier amount is 0, then the resultant interest/charge amount will be 0.

        Validation Rules: Should be a Valid record in AA.SOURCE.CALC.TYPE It an optional field, if this field left blank then the tire calculation based on default source type. Only applicable for Interest and Charge properties For an Interest CALC.PROPERTY the AA.SOURCE.CALC.TYPE defined here should have SOURCE.TYPE as 'Balance' and CALC.TYPE as 'Daily'. TIER.SOURCE.TYPE for the corresponding Interest condition will be applicable only when the Tier percentage is defined in the interest condition.
