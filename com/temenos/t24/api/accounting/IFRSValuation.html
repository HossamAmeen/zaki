        public TNumber getEffectiveInterestRate(List<CashFlow> cashFlows, InterestBasisSource interestBasisSource, TNumber initialApproximationRate)
        Calculates the Effective Interest Rate for the given cash flows using the specified interest basis source and initial approximation rate.
        T24 Details: This invokes the T24 routine EB.CALC.EIR.
        Parameters:
        cashFlows - The cash flows for the contract; amounts and dates
        interestBasisSource - Interest basis source details including the source table name INTEREST.BASIS or CURRENCY and the record id
        initialApproximationRate - The rate to seed the initial iteration for the Effective interest Rate calculation
        Returns:
        TNumber - The Effective Interest Rate.
        Throws:
        T24CoreException if parameters are not set or invalid.

        public TNumber getNetPresentValue(List<CashFlow> cashFlows, String interestBasisId, String currencyId, String rateKey, Margin marketRateMargin, String rateType, TDate startDate)
        Calculates the Net present Value for the given cash flows using the specified interest basis id, periodic interest id.
        T24 Details: This invokes the T24 routine EB.CALC.NPV
        Parameters:
        cashFlows - The cashflows for the contract; amounts and dates
        interestBasisId - Interest basis id for periodic calculation
        currencyId - The currency id is used in the formation of periodic interest key. e.g. USD
        rateKey - The Rate Key is the first two digits of the periodic interest id e.g. 23
        marketRateMargin - Market rate Margin for the calculation
        rateType - The rate to use; one of BID or OFFER
        startDate - Start date for the calculation
        Returns:
        TNumber - The Net Present Value.
        Throws:
        T24CoreException if parameters are not set or invalid.
