        public TField getCorrectLcyExcep()
        Returns the CorrectLcyExcep

        Field description: Specifies if an exception occurs between the profit and loss and asset and liablilty balances, that the difference should be automatically corrected. If set to Y the system will raise the difference to the suspense account in local currency of the category defined in the CONSOLIDATE.COND record ASSET&LIAB in the field ADJ.ACCT.CAT. The posting will be made with the transaction code defined in the field ADJ.ACCT.TXN. The difference amount can be found in the field EXCEPT.AMT.LCY. Validation Rules: Optional Input Valid entry of Y only Entry of Y is only allowed if the record key is the last working day Entry of Y is only allowed if the EXECPT.AMT.LCY is not null
