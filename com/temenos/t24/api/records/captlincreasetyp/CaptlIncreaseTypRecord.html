      • getAdvice

        public TField getAdvice()
        Returns the Advice

        Field description: Specifies which format print is to be used to print the advices. The value you specify should point to a particular record on the DE.FORMAT.PRINT file for those advice types which relate to capital restructuring activities. These are located under advice type 1510. The key to the DE.FORMAT.PRINT records is similar to the following: 1510.1.1.GB 1510.2.1.GB Where 1510 relates to Captial Increases generally and the second part of the key defines the format. You should ensure that the format you wish to use exists as this cannot be validated at this point. For example, if the above two records only existed and version 3 was selected then NO advice would be created by the delivery system. This field will be defaulted into field ADVICE.PRINT.FLAG on the CAPTL.INCREASE.CUS record. Validation Rules: 1-5 numeric characters. (Optional input.)

        Returns:
