      • addFilterByProduct

        public void addFilterByProduct(java.lang.CharSequence param)
        Appends the specified FilterByProduct to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Certain Periodic Attributes are capable of evaluating the rule on multiple Arrangements (either based on associated arrangements from the Bundle or based on CRA). This is controlled by the Attribute Multi Arrangement in AA.PERIODIC.ATTRIBUTE. However, if there is a need to filter these multiple arrangements down to a sub-set based on the Products to which they belong, the same could be specified in this field. Before evaluating the Source on each of those Arrangements, system would filter that list down to only those belonging to the Product(s) specified here. Validation - Only valid AA.PRODUCT allowed

        Parameters:
        param - the FilterByProduct to append
        Returns:
        nothing
