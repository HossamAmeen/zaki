        public void setAvblMarker(TField value)
        Set the specified value for AvblMarker
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Field that indicates whether the Facility is available for drawings or not. In many instances there could be a need to temporarily suspend the facility for a certain time period. The bank would require suspension of facility in instances where documentation is pending (or to be renewed), need for additional collateral or existing collateral to be substituted etc. In such circumstances, the facility is withdrawn for a certain time period till such time the requirement is fulfilled.
