      • getHolRestrictType

        public TField getHolRestrictType()
        Returns the HolRestrictType

        Field description: Indicates the type of holiday restriction. Holds the following values 1. PropertyClass - indicates the holiday restriction is of property class type and all the properties belonging to the given property class will be eligible for holiday restriction. 2. Property - indicates the holiday restriction is of property type and only the given properties are eligible for holiday restriction. 3. BillType - indicates holiday restriction is of "Billtype" type. BillType, PaymentType of Bill Type and Properties of Bill Type in the payment schedule definition are eligible for holiday restriction only if the BillType given in field matches the counterpart in the payment schedule definition. 4. PaymentType - indicates holiday restriction is of "PaymentType" type. PaymenType and properties of Payment type in the payment schedule definition are eligible for holiday restriction only if the PaymentType given in this field matches the counterpart in payment schedule defintion. During Update Payment Holiday Activity, the BillType/ Payment Type in the activity is cross validated against these fields before proceeding with updating holiday information. Non-Mandatory Multi-Valued field.

        Parameters:
        Returns:
