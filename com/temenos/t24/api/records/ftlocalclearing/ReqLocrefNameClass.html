        public TField getReqLocrefName()

        Field description: Specifies the name of the LOCAL REFERENCE field required by the local clearing system. This gives the name of the local reference field required by the local clearing system. This name must be present on the FT.BC.PARAMETER record for the clearing system defined in field CLEARING.SYSTEM. The field that it must be present in on the FT.BC.PARAMETER record is LOCREF.ITEM.REQ. Validation Rules: 1-20 alphanumeric characters. Multivalue field associated with REQ.LOCREF.POS. (Optional input) Must be present in field LOCREF.ITEM.REQ on the FT.BC.PARAMETER file.
