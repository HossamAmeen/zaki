      • getCcyCompany

        public TField getCcyCompany()
        Returns the CcyCompany

        Field description: The company code for the local currency Curreny file when taking rates at report time from the Local cuurency Currency files. This field contains the Company code whose currency file is to be used when the exchange rates are to be taken from the Currency file for each local currency. Only needs to be entered when there is more than one currency file for the same local currency. Suppose that company C and company D both have the Local currency as DEM and for specific reason the exchange rates to be used in each company must be different. Then either company C's or D's exchange rates must be used when creating a combined report using exchange rates from a local currency file. For this to be done then DEM must be entered in the previous field LCL.CCY and either company C or D in this field. The values in this field are defaulted by the system when there is only one Currency file for each Local currency. Validation Rules: The 9 digit company code or the 3 digit company mnemonic. Multi value field associated with LCL.CCY. Optional input. Must be a valid Company code on the company file. The local currency for this company must be equal to the currency specified in the corresponding multivalue in the field LCL.CCY. A Company code must be entered when there is more than one Currency file for the Local cuurency in field LCL.CCY.

        Parameters:
        Returns:

      • getLclCcy

        public TField getLclCcy()
        Returns the LclCcy

        Field description: The local currencies for the companies in the field COM.CONSOL.FROM. This field contains a list of the local currencies from the companies to be combined together. Only needs to be entered when there is more than one currency file for the same local currency. Suppose that company C and company D both have there Local currency as DEM and for specific reason the exchange rates to be used in each company must be different. Then either company C's or D's exchange rates must be used when creating a combined report using exchange rates from a local currency file. For this to be done then DEM must be entered in this field and either company C or D in CCY.COMPANY. The system will default the list of the local currencies for this group. Validation Rules: The 3-digit currency code or the 3-digit numeric currency code. Multi-value field associated with field CCY.COMPANY. Optional input. Must be a valid Currency code on the currency file. Must be the local currency for one of the companies being combined in the report.
