        public void addConsolidateExpression(TField param)
        Appends the specified ConsolidateExpression to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Consolidate the multiple RULE.NAME expressions and define a consolidated rule. This is so that conditions from multiple formlets (contexts) could be combined. This would be used to cater to scenarios where a formlet is made mandatory in FORM.RULE based on conditions getting satisfied from multiple formlets.

        Eg. If the Rule Names are NON.IRISH and DATA.INPUT.COMPLETED, the Consolidated expression can be like NON.IRISH AND DATA.INPUT.COMPLETED
