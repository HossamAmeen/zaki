      • getMvmtTarget

        public TField getMvmtTarget()
        Returns the MvmtTarget

        Field description: The target balance associated with the event type. This field is sub-valued and can be expanded to allow multiple movements to be raised. The values allowed when MVMT.DETAIL is set to STMT are: BAL*BALANCE.TYPE key Soft Balance Type (Defined in AC.BALANCE.TYPE) INT*ACCOUNT.CLASS key AccountClass for posting to an internal account of the specified class in event currency INTLCY*ACCOUNT.CLASS key AccountClass for posting to an internal account in local currency of the specified class. AC*ACCOUNT key account number for posting to a specific account number. TXN* Contract id to be mapped from the supplied event information The values allowed when MVMT.DETAIL is set to CATEG are: PL*CATEGORY key category code for profit and loss posting The values allowed when MVMT.DETAIL is set to SPECIAL are: CRB TXN indicating that the contract of the event will be used. No Input allowed if the MVMT.DETAIL is not populated.

        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: The target balance associated with the event type. This field is sub-valued and can be expanded to allow multiple movements to be raised. The values allowed when OPP.DETAIL is set to STMT are: BAL*BALANCE.TYPE key Soft Balance Type (Defined in AC.BALANCE.TYPE) INT*ACCOUNT.CLASS key AccountClass for posting to an internal account of the specified class in event currency INTLCY*ACCOUNT.CLASS key AccountClass for posting to an internal account in local currency of the specified class. AC*ACCOUNT key account number for posting to a specific account number. TXN* Contract id to be mapped from the supplied event information The values allowed when MVMT.DETAIL is set to CATEG are: PL*CATEGORY key category code for profit and loss posting The values allowed when OPP.DETAIL is set to SPECIAL are: CRB TXN indicating that the contract of the event will be used.
