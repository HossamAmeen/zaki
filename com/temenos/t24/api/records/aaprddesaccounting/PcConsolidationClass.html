      • getPcConsolMethod

        public TField getPcConsolMethod()
        Returns the PcConsolMethod

        Field description: Indicates the consolidation method that applies to the associated Charge / Tax Property class. When this field is left blank, when Consolidation field is defined, value will be defaulted as 'Net' for Charge property class and as 'Itemised' for Tax property class.

        Allowed Values:

        1. Net - When Consolidation Method is set to 'Net' for Charge property class the net of the calculated amount and the waived amount will be generated as a single entry. When Consolidation Method is set to 'Net' for Tax property class the accounting will be done only for the net of tax amount.

        2. Itemised - When Consolidation Method is set to 'Itemised' for Charge property class, two separate set of entries will be raised. One for Charge calculated amount and another for waived amount. When Consolidation Method is set to 'Itemised' for Tax property class, two separate set of entries for Tax and Calculated interest / charge amount will be raised.

        Parameters:
        Returns:
        TField - The PcConsolMethod
