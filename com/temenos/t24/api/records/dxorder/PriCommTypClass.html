        public TField getPriCommCde()
        Returns the PriCommCde

        Field description: If SCDX.CHARGE.METHOD is enabled in DX.PARAMETER, This will be defaulted with Commission code / tax code with which CHARGE.TAX.AMT is derived. Validation Rules: Up to 20 characters &#150; input must exist on FT.COMMISSION.TYPE or FT.CHARGE.TYPE No Input field unless there is input to PRI.COMM.TYP
