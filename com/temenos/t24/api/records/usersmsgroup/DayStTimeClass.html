      • getDayEndTime

        public TField getDayEndTime()
        Returns the DayEndTime

        Field description: If the same field in USER record is empty, then the value in this field will be used. Defines the end period or the log out time till which the user is allowed access to the system on a specific day defined in the ALLOWED.DAYS. Input can be 0-2359 or 0000-2359 or 00:00-23:59 Note: (1) This field is always displayed formatted with ':' (2) For whole day/night login, start should be 00:00, end should be 24:00 (ie. 24 hours). (3) There may be two or more separate periods each day when the User is allowed to access the System, e.g. morning and afternoon, but not during lunchtime. (4) If only 1 or 2 digits are entered, they correspond to minutes between midnight and 1 o'clock in the morning. Validation Rules: 1-4 numeric characters with optional ':' i.e. HHMM or HH:MM Sub-value field associated with ALLOWED.DAYS and DAY.ST.TIME (Optional input)

        Returns:
