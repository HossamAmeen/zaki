      • getUnplugDate

        public TField getUnplugDate()
        Returns the UnplugDate

        Field description: The unplug date is part of a multi value set linked to the START.DATE. The unplug date is the last date from which a portfolio can be included in a composite performance. The unplug date is automatically calculated by the system based on a the CLOSE.DATE and the PLUG.DATE. The unplug date cannot be manually amended. e.g. if a portfolio is removed from a composite on the 16th July 2000 and the plug date is the 1st January 2000. The unplug date will be the 1st July 2000. Validation Rules: Standard date format, mandatory field generated by T24. The unplug date must be greater than or equal to the plug date.
