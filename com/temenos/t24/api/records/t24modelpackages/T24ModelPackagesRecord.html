        public TField getUpdParamType()
        Returns the UpdParamType

        Field description: This field is to store the parameter option used with T24PackageInstaller parent JBC routine when installing respective pacakges Validation Rules: Valid values : DataOnly - package installer triggered with parameter option 'DataOnly' to deploy data records from package LocalPack - package installer triggered with parameter option 'LocalPack' to deploy both object codes and data records from package , object code gets deployed into local object directory defined in DS.PARAMETER SYSTEM record if DS installed.

        Blank if package installer triggered without any parameter options to deploy both object code and data records from package, the object code gets deployed into t24lib and t24bin

        Returns:
