        public TField getNewCycleName()
        Returns the NewCycleName

        Field description: Specifies the name by which this fiche cycle will be identified. The name entered here will be used to identify all the records selected for this fiche cycle from files F.FICHE.HOLD.CONTROL and F.FICHE.DE.HEADER. When the records have been written to tape the cycle name entered here will be moved to field LAST.CYCLE.NAME and the records identified by the previous entry in LAST.CYCLE.NAME will be deleted from the files. If another tape is to be produced of records selected by the LAST.CYCLE.NAME then the fields NEW.CYCLE.NAME, CUT.OFF.DATE and CUT.OFF.TIME should be left blank. If field LAST.CYCLE.NAME is blank then a new cycle name, cut off date and time should be specified. Validation Rules: 0-20 type A (alphanumeric) characters.

        Field description: Specifies what output (reports or delivery) is to be saved to tape.
