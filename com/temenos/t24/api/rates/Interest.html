com.temenos.t24.api.rates

Class Interest

Packaged in : ST.InterestApi.jar

Provides access to various methods for calculating different types of interest rates or periods. All methods in this class require the interestId to be set, using setInterestId().

      • calculateAccrualDays

        public TNumber calculateAccrualDays(TDate startDate, TDate endDate, String interestDayBasisId)
        Calculates the number of interest accrual days between two given dates using the interest day basis.
        T24 Details: This invokes the T24 routine BD.CALC.DAYS. The interest day basis type used by this routine is defined in the T24 table INTEREST.BASIS.
        Parameters:
        startDate - Start date of the interest accrual period. The start date is always included in the calculation.
        endDate - End date of the interest accrual period. The endDate is always excluded from the calculation.
        interestDayBasisId - Required. Type of interest calculation, e.g A1, A4, H, W, X etc as per the INTEREST.BASIS table in T24.
        Returns:
        TNumber - the number of days for which interest was or will be accrued.
        Throws:
        T24CoreException if parameters are empty or invalid.

      • getBasicRate

        public TNumber getBasicRate(String currencyId, TDate interestDate)
        Returns the interest rate for the given currency on or near the given date. Each currency can have a different rate. If there is no exact match, the nearest date which is less than or equal to the interestDate will be used.
        T24 Details: This invokes the T24 routine EB.GET.INTEREST.RATE. Values come from the T24 table BASIC.INTEREST.
        Parameters:
        currencyId - Required. The Currency to get the rate for.
        interestDate - Required. Date on which the rate is queried.
        Returns:
        TNumber - the interest rate for the given parameters.
        Throws:
        T24CoreException if the currency is not defined for the given interestId.

      • getTermMarginRate

        public TermRate getTermMarginRate(String currencyId, String bidOrOffer, TNumber dealAmount, TDate startDate, TDate endDate, String margin)
        Returns the margin rate for the given currency and period.
        T24 Details: This invokes the T24 routine TERMRATE. Information is gathered from the T24 table MARGINS with rates calculated via linear interpolation as required.
        Parameters:
        currencyId - Required. Currency for which the rate is expected.
        bidOrOffer - Indicates whether a Bid or Offer rate is required. Values should be 'B' or 'O', defaults to 'B'.
        dealAmount - Optional. The amount to which the margin is to apply.
        startDate - Optional. Desired start date. The calculation wil use the first working day after startDate if startDate is a holiday. Defaults to the current business date if null.
        endDate - Desired maturity date. The calculation will use the first working day before endDate if endDate is a holiday.
        margin - Optional. The signed margin value applied to the rate for given value.
        Returns:
        TermRate - the rate of interest and the number of days between the given dates.
        Throws:
        T24CoreException if interestId, currencyId or endDate are not provided.

      • getTermMisRate

        public TermRate getTermMisRate(String currencyId, String bidOrOffer, TNumber dealAmount, TDate startDate, TDate endDate, String margin)
        Returns the MIS refinacing rate for a specified currency and period. The MIS rate is an internal reference rate used for management reporting of profitability for a transaction.

        T24 Details: This invokes the T24 routine TERMRATE. Information is gathered from the T24 tables PERIODIC.INTEREST and MARGINS with rates calculated via linear interpolation as required.
        Parameters:
        currencyId - Required. Currency for which the rate is expected.
        bidOrOffer - Indicates whether a Bid or Offer rate is required. Values should be 'B' or 'O', defaults to 'B'.
        dealAmount - Optional. The amount to which the margin is to apply.
        startDate - Optional. Desired start date. The calculation wil use the first working day after startDate if startDate is a holiday. Defaults to the current business date if null.
        endDate - Desired maturity date. The calculation will use the first working day before endDate if endDate is a holiday.
        margin - Optional. The signed margin value applied to the rate for given value.
        Returns:
        TermRate - the rate of interest and the number of days spotted between the given dates.
        Throws:
        T24CoreException if intrestId, currency or endDate are not provided.
