      • getMarketPlCat

        public TField getMarketPlCat()
        Returns the MarketPlCat

        Field description: Specifies the profit & loss category for any potential 'Market' profit or loss with an OPTION.TRADE. This code classifies profit and loss and product types. Each business transaction within T24 will be allocated a CATEGORY code which will enable reports to be constructed that give an accurate view of the bank's operations. Note: At the moment the customer price must equal broker price so there is no possible chance of market profit and loss occuring within an OPTION.TRADE. Validation Rules: 4 or 5 numeric characters. a) Product/P&L classification in the range 1-99. b) Sub-classification of product/P&L in the range 000-999. (Mandatory Input) Must exist as a valid CATEGORY code.

        Parameters:
        Returns:
        TField - The MarketPlCat

      • getMarketPlCrCode

        public TField getMarketPlCrCode()
        Returns the MarketPlCrCode

        Field description: The default TRANSACTION code for all Credit transactions within OPTION.TRADE where market profit & loss occurs. To facilitate the matching of the TRANSACTION code to the application to which it refers ranges have been assigned to each application. The following ranges have been assigned within T24: 600 - 649 = Securities Validation Rules: 1-10(Max Value) numeric characters Transaction Code. The Maximum value is specified in EB.OBJECT for TRANSACTION. Must exist as a valid TRANSACTION code.
