        public TField getDr2BasicRate()
        Returns the Dr2BasicRate

        Field description: Field range DR2.BASIC.RATE to DR2.LIMIT.AMT is used to specify rates applicable to different balance limits. Specifies a Base Rate which is held on the BASIC.INTEREST table (Ref: General Tables). (DR2 fields specify a second interest charge which may be levied.) Rates can be fixed or linked to Basic Rates plus or minus a Margin. For examples see DR.BASIC.RATE. Validation Rules: 1-4 numeric characters. Multivalue field associated with field range DR2.BASIC.RATE to DR2.LIMIT.AMT. (Optional input. Not allowed if corresponding DR2.INT.RATE is input or if INTEREST.DAY.BASIS contains NONE or GENERAL.) There must be a record in the BASIC.INTEREST table with a key consisting of this code, the Currency of the Account, and an effective date which is less than or equal to the date specified in ID.

      • getDr2MarginOper

        public TField getDr2MarginOper()
        Returns the Dr2MarginOper

        Field description: Field range DR2.BASIC.RATE to DR2.LIMIT.AMT is used to specify rates applicable to different balance limits. Determines the method by which the Rate specified in corresponding DR2.BASIC.RATE or DR2.INT.RATE is adjusted by the Margin specified in corresponding DR2.MARGIN.RATE. (DR2 fields specify a second interest charge which may be levied.) If the Rate specified in DR2.BASIC.RATE or DR2.INT.RATE is to adjusted by a Margin, this field specifies whether the Margin Rate is to be added to, subtracted from, or multiplied with the basic rate to arrive at the effective interest rate. If Multiply is specified the final rate is (100 + M) percent of the Rate from DR2.BASIC.RATE or DR2.INT.RATE where M is the Margin Rate percentage from DR2.MARGIN.RATE. For examples see DR.BASIC.RATE. Validation Rules: ADD, SUBTRACT or MULTIPLY, or any abbreviation thereof. Multivalue field associated with field range DR2.BASIC.RATE to DR2.LIMIT.AMT. (Optional input. Only allowed if corresponding DR2.BASIC.RATE or DR2.INT.RATE is input.)

        Parameters:
        Returns:
        TField - The Dr2MarginOper

        public TField getDr2MaxRate()
        Returns the Dr2MaxRate

        Field description: This field defines the capped rate (or maximum rate) for debit interest to be applied to the DR2 balance type. If the rate that is derived by T24 after applying any spread, is more than this rate then the interest rate will be capped at this value. Validation Rules: Rate field.
