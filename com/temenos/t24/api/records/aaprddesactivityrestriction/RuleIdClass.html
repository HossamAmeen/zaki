      • getRestrictType

        public TField getRestrictType()
        Returns the RestrictType

        Field description:

        The type of restriction that need to be applied on the activity ERROR - The transaction triggering this activity would be completely blocked. OVERRIDE - The transaction triggering this activity would be allowed after displaying a warning.

        1. Will accept either OVERRIDE or ERROR 2. If OVERRIDE is specified then RESTRICT.OVR is mandatory 3. If ERROR is specified then RESTRICT.ERROR is mandatory

        Parameters:
        Returns:
        TField - The RestrictType
