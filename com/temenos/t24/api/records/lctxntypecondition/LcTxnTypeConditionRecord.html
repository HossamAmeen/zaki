        Field description: Describes the charge type in an abbreviated format which can be used for reporting where space is limited. This multivalue field is used to enter a Short Description for the Charge Type code in various languages. Each expansion will correspond to a new language as defined in the Language Table and will indicate to the user the language in which the short description must be entered.
