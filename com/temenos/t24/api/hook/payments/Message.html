        public void updateInformationLine(List<CreditParty> partyCredit, List<DebitParty> partyDebit, List<coverInformation> coverInformation, DebitAuthorityInformation debitAuthorityInformtion, HeaderInformation headerInformation, RemittanceInformation remittanceInformation, List<HistoryLog> historyLog, DirectDebitInitiation directDebitInitiation, PaymentFlowInformation paymentWorkFlowInformation, List<StatusInformation> statusInformation, TransactionInformation transactionContext, List<InformationLineDetails> informationLine, List<AdditionalInformationLineDetails> additionalInformation)
        This interface is used to update the information and additional information lines.
        The interface is invoked during the mapping stage of the message.
        T24 Details: The EB.API hook used by this interface is MAPPING.PARAMETER.ENRICH.API.HOOK This interface is invoked by the routine T24MessageMappingServiceImpl.doMessageMappingSingle,InwardMappingFramework.executeMessageLevel and T24InwardCreditTransferInitiationServiceImpl.createTransaction
        * The T24 field specifying this hook is the Enrichapi field in PP.MSGMAPPINGPARAMETER.
        If an exception is thrown in the implementing class, the exception message will be displayed on screen and no results will be displayed.
        Parameters:
        partyCredit - It contains the information about credit party
        partyDebit - It contains the information about debit party
        coverInformation - It contains information about cover message
        debitAuthorityInformtion - It represent the information about debit authority
        headerInformation - It contains information about header
        remittanceInformation - It contains information regarding remittance
        historyLog - It has the information related to history table
        directDebitInitiation - It contains information about direct debit
        paymentWorkFlowInformation - It contains informations about payment work flow details
        statusInformation - It has details regarding status
        transactionContext - It contains all information related to transaction
        informationLine - This is a list of Information Details.
        additionalInformation - This is a list of additional information
        Returns:
        nothing
