        public TField getAccPortNumber()
        Returns the AccPortNumber

        Field description: This field will have any one of the below values I. Account number II. Portfolio number III. MM contract ID IV. LD contract ID Fields starting from ACC.PORT.NUMBER to ACC.BAL.USD are associated fields. Each muti value set will hold the balance details of each ACCOUNT/MM Contract/LD Contract linked to the customer. When this field holds a portfolio number, all the security position of the portfolio will be aggregated in a single multi value set.
