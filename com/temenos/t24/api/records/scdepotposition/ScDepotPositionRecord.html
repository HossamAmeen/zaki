      • getInterestRate

        public TField getInterestRate()
        Returns the InterestRate

        Field description: This field is only updated is this security position is for a Kassenobligationen security. A Kassenobligationen security is one where the Sub Asset Type record linked to the security has the KASSENOBLIGATIONEN flag set to YES. If the security identified in the SECURITY.NUMBER is a Kassenobligationen security then this will be the interest rate for the security - taken from the INTEREST.RATE field on the Security Master record - at the time the SC.DEPOT.POSITION record was created. The Interest Rate (if any) in this field will make-up the elements that make up the ID of the SC.DEPOT.POSITION file.

        Parameters:
        Returns:
