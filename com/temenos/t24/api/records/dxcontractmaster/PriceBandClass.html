        public TField getPriceBand()
        Returns the PriceBand

        Field description: Some contracts are subject to a price band, specified on the contract specification set by the relative Exchange. This field works in conjunction with Tick Size, Tick Value, and Minimum Price Movement. An example of price band contract is the 3 Month Short Sterling Option; <7(note not equal too) price units the Tick Size is 0.01, Tick Value is 12.50(6.25 1/2 tick) and the minimum price movement is 0.005 >7 price units the Tick Size is 0.01, Tick Value is 12.50 and the minimum price movement is 0.01 This equates; the movement up to 7 price units will be per half tick and above 7 price units per one tick. The set is then multi-valued where the, Validation Rules: Up to 16 characters available in "CURRENCY'" format or Alpha "OPEN" for no price band. Multi-value Set with Tick Size, Tick Value, Minimum Price Movement and Price Tolerance. Price Band, field is shown as an "Up Too" figure. No Change field
