        public void addDividendTrans(TField param)
        Appends the specified DividendTrans to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: List of securities transaction types that the user wishes to define as to default the accounts previously input in the DIV.ACCOUNT.NO field. Will be used by the conversion program CONV.SEC.ACC.MASTER.14.2 to set-up the portfolio account defaulting. The conversion program will take the accounts previously entered in the DIV.ACCOUNT.NO field and set-up the transaction types input into this field so that the accounts will default when these transaction types are used for the portfolio concerned. Validation Rules: Must exist on the SC.TRANS.NAME file.
