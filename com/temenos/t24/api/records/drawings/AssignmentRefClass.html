      • getAssignmentRef

        public TField getAssignmentRef()
        Returns the AssignmentRef

        Field description: Assignment release given in LC. On entering the assignment reference it will be checked with the reference given in LC and all related information will be brought to DRAWING record if the reference matches. If value 'TPPAY' is input, then it will be understood as multiple settlement to third parties. In such case, updation, verification of LC record is not done. If value 'TPRECV' is input, then it will be understood as multiple debits to Applicant/Third parties. In such cases, update, verification of LC record is not done. Validation Rules: Mandatory input if assignment or third party settlement is to be done. Allowed for Import and Export transactions. Only the reference used in LC or TPPAY alone will be accepted for the credit leg. Only TPRECV can be used if multiple debits are to be raised.

        Parameters:
        Returns:
        TField - The AssignmentRef
