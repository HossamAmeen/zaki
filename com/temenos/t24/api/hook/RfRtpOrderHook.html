        public void getResultOption(String paymentOrderId, PaymentOrderRecord paymentOrderRecord, String sourceSystem, TString resultOption, TString errorMessage)
        This interface enables the implementer to determine the result option which in turn will be used for mapping back to the source system.
        This interface is invoked by routine PAYMENT.ORDER.AUTHORISE and PAYMENT.ORDER.NOTIFICATION.AUTHORISE.
        The EB.API hook used by this interface is PI.SOURCE.MAP.RULE.API.HOOK
        The T24 field specifying this hook is SOURCE.MAP.RULE.API in PI.PS.MAPPING.RULE.
        If an exception is thrown in the implementing class it will be treated as an error.
        Parameters:
        paymentOrderId - Id of the payment order record.
        paymentOrderRecord - Payment Order record.
        sourceSystem - source system
        resultOption - result option
        errorMessage - Error Message with type string.
        Returns:
        nothing
