        public TField getCipherMethod()
        Returns the CipherMethod

        Field description: To specify an Algorithm to be used for digital signature signing (by front end) and verification (at t24 layer). Input allowed only for SYSTEM record. This field accepts list of valid names pre-defined in field property and T24 will convert it into equivalent algorithm name (during run time) for digital signature verification. i.e Input value = GENERAL , real algorithm name = JBASE_CRYPT_GENERAL. Validation Rules: Mandatory field for SYSTEM record. Valid values are: GENERAL, XOR11, RC2, DES, 3DES, BLOWFISH, AES-BASE64, RC2-BASE64, DES-BASE64, 3DES-BASE64, BLOWFISH-BASE64 NOCHANGE field.
