      • getGraceDays

        public TField getGraceDays()
        Returns the GraceDays

        Field description: This is a Numeric field and shall be used to input to indicate the additional number of days for which the interest on the discounted drawing is to be recovered. This amount thus calculated for the number of days input in this field, will be in addition to the Discount and Load amount calculated. This field thus represents the cushion that could be made available in the event of the settlement being delayed for a Discounted Drawing. This shall however have no impact on the Maturity Date of the Drawing. Amortization of amount thus collected shall take place only till the MATURITY.REVIEW date or the NEW.MATURITY.DATE as input. In the event of such Discounted Drawing being extended using DR.DISC.AMENDMENTS, the amortization shall take place till the NEW.MATURITY.DATE. However, if the same is matured online or back-dated, then the amortization would have taken place only till such date.The input in this field is permitted only at the first instance of discounting a Drawing. The field opens up only if the DiscountLoad rate is input.

        Parameters:
        Returns:
        TField - The GraceDays
