        public TField getExcRuleAppl()
        Returns the ExcRuleAppl

        Field description: Field to specify the name of the application, which are required to be excluded from the balance aggregation process. Fields from EXC.RULE.APPL to EXC.RULE.VALUE are associated multi-value set. Each multi-value set of the selection criteria will be considered with 'OR' condition. Validation rules

        Currently SEC.ACC.MASTER alone is allowed to define in this field and the portfolios selected from this selection criteria are exempted from balance aggregation.

        EXC.RULE.APPL, EXC.RULE.FIELD, EXC.RULE.OPERAND, EXC.RULE.VALUE are interdependent, all must be populated or blank.
