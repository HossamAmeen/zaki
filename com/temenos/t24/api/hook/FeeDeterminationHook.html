        public TNumber getChargeAmount(PaymentDetails transactionDetails)
        This interface enables the implementer to get the charge amount for the transaction(Unconditional Fee).
        This interface is invoked by the routine FeeDeterminationService.calculateCharges.
        The EB.API hook used by this interface is FEE.TYPE.FEE.API.HOOK.
        The T24 field specifying this hook is the Feeapi field in PP.FEETYPE application.
        If an exception is thrown in the implementing class it will be treated as an error.
        Parameters:
        transactionDetails - It contains the details like transaction amount,fee amount etc
        Returns:
        TNumber - Charge Amount.
