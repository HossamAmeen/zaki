        public TField getCreateDimItems()
        Returns the CreateDimItems

        Field description: When the user creates a new property class and artefacts for them, and requires all these artefacts to be chucked into PvCS, the user may decide to flag this field which updates all items updated to DIM.NEW.ITEMS record. The id of DIM.NEW.ITEMS application would be taken from DIM.NEW.NAME. If a record already exists for this id, then the current details would be appended onto the existing record. The user would still have to verify the DIM.NEW.ITEMS record to create the actual items. This is only a utility to create such record.
