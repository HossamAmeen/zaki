        public TField getAutoShutdown()
        Returns the AutoShutdown

        Field description: Specifies whether the system should attempt to automatically shut down the EB.PHANTOM process prior to running the batch. If this field is set to Y the system will try to close the phantom process if the STATUS is ACTIVE. If the process cannot be closed the system cannot run the end of day. In order to correct this situation the operator should determine whether the process is actually ACTIVE, if not the STATUS flag should be reset manually. Validation Rules: Optional field, valid value Y.
