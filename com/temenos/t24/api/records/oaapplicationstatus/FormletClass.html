      • getFormletAttribute

        public TField getFormletAttribute()
        Returns the FormletAttribute

        Field description: This field would indicate whether Formlet is mandatory or Optional to complete before submitting the form. All the formlets which are defined as mandatory at OA.DOMAIN.TYPE level would be flagged as MANDATORY in this field. The formlets which are become mandatory because of the Conditional formlets Rule definition also would flag as MANDATORY once the Rule has been satisfied. The optional formlets would have the null value. The system will not allow the user to submit the form until all MANDATORY formlets are recorded and complete status.
