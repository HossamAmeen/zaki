      • getPeriod

        public TField getPeriod()
        Returns the Period

        Field description: Indicates the actual period of constraint. The various options of the field are: If TYPE is stated as STATEMENT.PERIOD, DATE, FINANCIAL.YEAR, RENEWAL - no input to this field would be allowed. If TYPE is stated as INTEREST.PERIOD - Allows a numeric value in this field (for example, no backdating prior to 3 interest periods). If nothing is stated, system would not allow any backdating prior to current interest period by default If TYPE is stated as PERIOD – A valid IN2PERIOD value should be stated in this field. The field is mandatory with this option

        Cannot be stated if no option is chosen in TYPE field. Validation: Optional field Cannot be stated if TYPE field is not set. Valid Numerical value or a Period allowed based on the TYPE stated

        Returns:

      • getType

        public TField getType()
        Returns the Type

        Field description: Indicates the constraint type. The options allowed are: INTEREST.PERIOD – Activity would be restricted and governed by the Interest period of the arrangement. For example, to restrict any activity past the billing period, this option may be used. STATEMENT.PERIOD – Constraint would be based on when the statement was last generated for this account. When multiple statements are generated(including special ones), the latest statement date them would be taken and any date prior to that would be restricted. RENEWAL – When this option is chosen, any activity prior to the last renewal date would be restricted PERIOD – User could state a specific period(say 1M, 6M etc) to indicate specific periods before which the activity would be restricted DATE – Indicates a date prior to which back dating would be restricted FINANCIAL.YEAR - When this option is chosen, any activity prior to the last year end date would be restricted

        Validation Rules Non Mandatory field. Valid options are INTEREST.PERIOD, STATEMENT.PERIOD, RENEWAL, PERIOD, FINANCIAL.YEAR and DATE

        Parameters:
        Returns:
        TField - The Type
