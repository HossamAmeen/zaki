      • getControlMaster

        public TField getControlMaster()
        Returns the ControlMaster

        Field description: Nature of property control for the associated property / property class in Master Arrangement. Suppress – The Associated Property / Property Class will be suppressed in the Master Arrangement ( will be available in future). Replace – The Associated Property / Property Class condition in master arrangement will be replaced with the condition defined in PROPERTY.CONDITION field. Merge – The Associated Property / Property Class condition will be merged with the condition defined in PROPERTY.CONDITION field. This option will be supported only for Pricing Rule and Periodic Charges. This option will be restricted for Charges. Blank – No Processing done.

        Parameters:
        Returns:
