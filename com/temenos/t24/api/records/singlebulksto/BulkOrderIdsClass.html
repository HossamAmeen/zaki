        public TField getBulkOrderIds()
        Returns the BulkOrderIds

        Field description: Identifies which Bulk Standing Order is to be executed. It must be a valid ID in the Live Bulk Standing Orders file. The contents of this field must correspond to a valid entry on the Bulk Standing Order file, this being Field 0 BULK.STO.NO. Validation Rules: Up to 24 type A (alphanumeric) characters as follows: - The first part of the ID identifies the Account Number for which the Bulk Standing Order has been defined. - The second part of the ID is separated from the Account Number by a '.' and will be a sequential number. Associated multivalue field. (Mandatory input.)
