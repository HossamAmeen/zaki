      • getShortcutButton

        public TField getShortcutButton()
        Returns the ShortcutButton

        Field description: Defines the shortcut button and corresponding shortcut key values to be used by screens in the Browser product. Together with the corresponding shortcut key value, these pairs define a 'shortcut' method to access action keys on Browser screens. A number of pre-defined shortcut buttons are presented in this field that can then be associated with a single character key to be used as a shortcut to the button. For example, selecting a shortcut button of "COMMIT" and a corresponding shortcut key value of "C" will cause the "Commit" button to gain focus on a contract screen when the "Alt" key is pressed in conjunction with the "C" key. Then pressing the "Enter" key will cause the commit to take place. Validation Rules: The following values will be accepted: AUTHORISE = Authorise a contract.

        Parameters:
        Returns:
