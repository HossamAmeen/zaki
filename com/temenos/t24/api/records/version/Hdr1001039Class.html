        Field description: Enables the first line of headings for columns 01 to 39 of the screen or printed report to be specified. If the standard Version of the Application displays information on header line 2, the heading defined in this field will be overwritten. Validation Rules: Up to 39 (type A) alphanumeric characters including required spaces. Multivalue field associated with Fields 5 to 12. (Optional input. No default value.)
