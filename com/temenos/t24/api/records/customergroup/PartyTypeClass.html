        public TField getCustomerId()
        Returns the CustomerId

        Field description: This field is used to define the CUSTOMER or PERSON.ENTITY who are part of this group. If ORIG.PARTY is "Customer" must be a valid record on T24 Customer application. If ORIG.PARTY is "Person " must be a valid record on the PERSON.ENTITY application which has been classified as a PERSON record. If ORIG.PARTY is "ENTITY" must be a valid record on the PERSON.ENTITY application which has been classified as an ENTITY record. Validation Rules:

        Mandatory Input.

        Must be a valid customer record id.

        Customer entered should not be part of any other customer group or Liability group or Limit sharing group

        Duplicates not allowed
