        public TField getDiscPercent()
        Returns the DiscPercent

        Field description: Customer discount percentage. This is the percentage discount to be taken for SEC.TRADE transactions forcustomers linked to this SCTR.GROUP.CONDITION record. It will automatically feed through to the CU.DISC.PCENT fieldon any SEC.TRADE transaction. At present this field is only used by the SEC.TRADE and MF.TRADE applications andtherefore input will not be allowed for other specific applications. Validation Rules: Mandatory input if the associated TRADE.ACTIVITY field is SEC.TRADE. Input is allowed only if the TRADE.ACTIVITY is SEC.TRADE, MF.TRADE or ALL. Any input must be in the range 0 to 100.

      • getSecType

        public TField getSecType()
        Returns the SecType

        Field description: Identifies the level for calculating commissions within each trading activity. For each trade activity the user may define several security types in order to calculate the commission amount. i.e. Valid security number, Asset types prefixed with "A-", Sub Asset types prefixed with "S-" or "ALL" for allother types not defined. All these must exist on the trading charge parameter table. In addition to the above a suffix of '*' followed by a country code can be added. This will be used to calculatedifferent charges for different Stock Exchange Calculation Countries. In addition to the above a suffix of '*' followed by a valid channel from EB.LOOKUP>SC.CHANNEL can be added.This will be used to calculate different charges based on mode of channel a transaction is placed Validation Rules: Input is Mandatory Must exist on SCTR.CHARGE.PARAMETER.

        Parameters:
        Returns:
        TField - The SecType
