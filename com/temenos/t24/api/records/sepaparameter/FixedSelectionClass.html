      • getLoadRoutine

        public TField getLoadRoutine()
        Returns the LoadRoutine

        Field description: This field specifies the Routine executed after the selection of incoming files on the associated library or to make a fine transaction selection on the SEPA.OUTWARD.PENDING buffer file prior to producing an outgoing file on the associated library. In ‘INWARD’, this optional routine is executed after the selection of the incoming files on the selected library. Its purpose is to modify the common variables of the I_SEPADIR insert (e.g. remove a file from the list after a given time on the day or make specific checks and/or updates on a file prior to its treatment). If the routine returns a value in ETEXT, the treatment will stop and its content will be reported to the SEPA.INWARD.LOG file with the highest error level (2). If the routine returns a value in COMI, its content will be recorded in the same log file with the lowest comment level (0) and the treatment will continue normally.

        Validation rule Value upto 35 type ANY (Any Character). Associated Multi Value subset of field FILE.DIRECTION to DEFAULT.FTTC

        Parameters:
        Returns:
