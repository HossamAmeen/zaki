Package com.temenos.t24.api.hook.contract

  • Class Summary 
    Class Description
    CalculationThe CalculationHook component allows access to routines for customising the rates, tax and sweep amount etc.
    DirectDebitThe methods in the class enable the implementer to read information from the clearing file and default/amend the values in the DD.DDI application.
    FundsTransferThe ContractHook component allows access to routines for customizing the FT related record.
    LetterOfCreditThe LetterOfCreditHook component provides the interfaces, which enable a developer to provide their own logic to calculate and return discount rate, charges and perform validations.
    PositionThe contract.Position class provides the interfaces which enable a developer to provide their own logic to calculate and return positions.
    StandingOrderThe ContractHook component allows access to get status of the FT generated by Standing Order.
    SwapThe contract.Swap class provides the interfaces which enable a developer to extend the behaviour for swap related activities.
