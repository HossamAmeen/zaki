      • getSweepAccount

        public TField getSweepAccount()
        Returns the SweepAccount

        Field description: This field will hold the account to / from which the amount to be donated or swept. This field will be updated only for AR accounts with Component Credit check set up SWEEP.ACCOUNT is updated when there is an overdraft on the transaction amount for the account having retail sweeping set up and any of the donor account has enough funds to support the deficit. If the SWEEP.ACCOUNT is a donor account then the amount that the ECB account has to be received is updated in PROJECTED.CR.AMT and if the SWEEP.ACCOUNT is a receiver account , then the amount that the ECB account owes is updated in PROJECTED.DR.AMT. This is a No input field.
