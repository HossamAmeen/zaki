Class CustomerPosition

Packaged in : ST.EnquiryHook.jar

        public List<CustomerPositionId> getContractIds(String customerId, String parentId)
        This interface enables the implementer to return the list of contract id's which will be taken forward by CUSTOMER.POSITION enquiry. This interface is invoked during both online and close of Business through the CUS.BUILD.POSITION.DATA. With the returned contract / application id's it will the input this to the hook attached in the CUSTOMER.POSITION>CUS.EXT.RTN T24 Details: Interface method for the USER.SELECT.RTN field in CUSTOMER.POSITION in T24. If an exception is thrown in the implementing class during online process it gets displayed in the browser with the prefix "EB-EXCEPTION.RAISED". It it is during COB, then it logs in T24 file "EXCEPTION.LOG.FILE"
        Parameters:
        customerId - Id of the customer
        parentId - Id of the parent or liable customer to the customer in customerId param
        Returns:
        List<CustomerPositionId> - the list of contract id to the calling program.
