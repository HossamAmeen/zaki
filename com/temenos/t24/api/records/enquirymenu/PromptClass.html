      • getLinkName

        public TField getLinkName()
        Returns the LinkName

        Field description: Defines the name of the enquiry/menu to be invoked for the selection prompt. An enquiry or menu name to be invoked for the selection prompt should be specified here. The enquiry name will be passed to application ENQUIRY.SELECT to invoke the enquiry and hence must exist as a record on the ENQUIRY file. The name of the menu to be displayed must exist as a record in this file (ENQUIRY.MENU). Validation Rules: 0-32 type A (alphanumeric) characters.

        Parameters:
        Returns:
        TField - The LinkName

      • getLinkType

        public TField getLinkType()
        Returns the LinkType

        Field description: Defines the type of routine i.e. enquiry or menu to be invoked for the selection prompt. The type specified here relates to the entry specified in LINK.NAME. M - Indicates that the link is to another menu. S - Indicates that it is an enquiry to be invoked via the ENQUIRY.SELECT application. Validation Rules: 1 alphanumeric character, 'M' or 'S'. M - The item to be invoked is another menu. S - The item to be invoked is an enquiry.
