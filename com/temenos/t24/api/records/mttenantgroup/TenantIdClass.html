      • getCobStyle

        public TField getCobStyle()
        Returns the CobStyle

        Field description: Defines the COB style for each tenant in the group. This value will be used during MT.TSA.SERVICE.CONSOLE authorization stage to decide on TSA service id. Validation Rules: Acceptable values are: 1. STANDARD :- denotes standard COB 2. COMPANY.GROUP :- denotes company group level COB, requires input in COMPANY.GROUP.ID field. 3. COMPANY :- denotes company specific COB, requires input in COMPANY.ID field. Input allowed only for COB level tenant grouping (TENANT.GROUP.LEVEL = COB). This field is mandatory if COMPANY.GROUP.ID or COMPANY.ID are set and when TENANT.GROUP.LEVEL is COB.

        Parameters:
        Returns:
        TField - The CobStyle

      • getTenantId

        public TField getTenantId()
        Returns the TenantId

        Field description: Defines a valid tenant id as part of the group. The TENANT.ID , COB.STYLE, COMPANY.GROUP.ID, COMPANY.ID and SERVICE.COMP.MNE fields are associated multi-value set. Only a LIVE type tenant can be added to a LIVE type group; and of course NON-LIVE tenant for a NON-LIVE group. Validation Rules: Must be valid entry in MT.TENANT table. The TENAT.GROUP.TYPE must equate to the TENANT.TYPE.GROUP (from TENANT.TYPE) of that tenant (LIVE or NON-LIVE). Maximum of 35 characters allowed. Mandatory field.

        Returns:
