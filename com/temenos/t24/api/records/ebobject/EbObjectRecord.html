      • getMaxLength

        public TField getMaxLength()
        Returns the MaxLength

        Field description: Input is the maximum number of Characters to be used as an ACCOUNT ID. This field is also used for Alternate Key processing for SECTOR and INDUSTRY applications. Maximum length of alternate key for SECTOR, INDUSTRY should be specified here. The SECTOR and INDUSTRY fields in applications will get resized to the length specified here. This can be of any length irrespective of the current key field length defined in the application program. This will allow the actual alternate value input to be accepted by the lowest level validation routines. Alternate key max length for ACCOUNT, CUSTOMER will be referred from ALT.KEY.MAX.LENGTH field in EB.ALTERNATE.KEY. Used for formatting purposes only. Validation Rules: None, but should be less than 26.

        Returns:
