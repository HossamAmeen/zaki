      • getVclReference

        public TField getVclReference()
        Returns the VclReference

        Field description: Multi valued field containing the IDs of the VAULT.CONTROL records linked to this expectation record. Associated with the fields NO.NOM.RECD.DEL, NO.NOM.REF.NO, NO.NOM.REF.DATE, NO.NOM.DATE and NO.NOM.NARR. If the DIRECTION of VAULT.EXPECT record is DELEIVERY then this field allows the user to select record from the VAULT.CONTROL file to satisfy the number of securities that need to be delivered. If the DIRECTION is RECEIPT then this field will be automatically updated by the system with the next ID for the VAULT.CONTROL file as new VAULT.CONTROL records are created in the Vault.

        Parameters:
        Returns:
        TField - The VclReference
