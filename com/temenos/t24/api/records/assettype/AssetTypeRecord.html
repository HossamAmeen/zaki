        public TField getAdjMargin()
        Returns the AdjMargin

        Field description: This field is used to specify the Adjusted margin rate for calculating ADJ.MARGIN.AMT in SC.POS.ASSET.

        This field can either be used to specify the Low Advance Ratio for collateral calculations in Advance Collateral process or to specify the Diversified Margins for portfolios flagged for Diversification.

        If the field "CO.MV.CHECK" in SC.PARAMETER is set to "YES" to enable Preferential LTV functionality, Adj Margin will be used only as Low Advance Ratio in Advance Collateral process and will not be used for calculating Diversified Margin. In this case ideally Adj Margin should be lesser than the Margin Rate. Thus an override will be raised if Adj Margin is greater than Preferential Margin Rate/Standard Margin Rate.
