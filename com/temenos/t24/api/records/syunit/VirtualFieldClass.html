      • getFunction

        public TField getFunction()
        Returns the Function

        Field description: For the “Drill Down” function the PRI.VAULE and SEC.VALUE fields are ignored and the DRILLDOWN.ID, DRILLDOWN.APP and DD.FIELD.NAME fields are used to obtain a value from a T24 table. For all other functions this is the Function to be applied between PRI.VALUE and SEC.VALUE. There categories are Mathmatical, Boolean and String. It is the users responsibility to ensure that the operators are applied to the correct data types. Can be one of the mathematical functions: '+', '-', '*' or '/' One of the boolean functions 'EQ', 'NE', 'GT', 'LT', 'GE', 'LE', 'OR' or 'AND' The string function 'Concatenate' The assignment function '=' 'Drill Down' or null.
