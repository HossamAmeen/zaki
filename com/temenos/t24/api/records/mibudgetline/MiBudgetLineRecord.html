      • getSysRebuildBudget

        public TField getSysRebuildBudget()
        Returns the SysRebuildBudget

        Field description: Whenever the MI.BUDGET.LINE record is authorised this field is set to 'Y' and the MI.BUDGET file cleared of any records which would have been created by the old record. This flag indicates that the MI.BUDGET records need to be created from this record. When the database specified in the key is built, the build routine will check this file for any records pertaining to the database with this flag set. If so the MI.BUDGET records will be created and this flag set to null. Validation Rules: System generated. No input.

        public TField getUpLoadFile()
        Returns the UpLoadFile

        Field description: This field holds the full Unix pathname of the file which will be uploaded from disk onto the unathourised file when the record is verified and the up load option taken. The user should ensure that he or she has sufficient access to the pathname specified to allow the record to be read. Once complete, the record will have the status set to HLD. The file must be in a comma delimIted format otherwise unexpected results may occur. For this reason it is recommended that a template is created by down loading the record using the download option (again from the Verify function).
