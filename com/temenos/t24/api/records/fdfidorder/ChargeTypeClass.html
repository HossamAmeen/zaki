        public TField getChargeType()
        Returns the ChargeType

        Field description: Defines the type(s) of charge to be debited from the customer. The customer can be charged a fee at any time according to the associated CHARGE.EVENT if required. Any charges to be applied should be entered in this field. Charge types can be defined on either FT.COMMISSION.TYPE or FT.CHARGE.TYPE. Charges can be optionally defaulted from the applicable FD.GROUP.CONDITION record, if the field DEFAULT.CHARGES is set to Y on the applicable FD.CHARGE.PARAMETER record. Validation Rules: 2-11 alphanumeric charge type. (Optional input)
