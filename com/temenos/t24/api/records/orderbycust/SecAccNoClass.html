      • getValue

        public TField getValue()
        Returns the Value

        Field description: Sub-valued field associated with the field SEC.ACC.NO to CASH.BALANCE to give details of orders. This field contains a different value depending on whether this segment of the total order refers to a portfolio buying or selling securities. If this refers a portfolio buying a security then it is the total value of the portfolio in the security currency of the security being bought. If this refers to a portfolio selling a security then it is the current nominal holding of the security being sold by the portfolio. For TARGET order types this will be the current nominal holding. Validation Rules: No-Input field for information only.

        Set the specified value for Value
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Sub-valued field associated with the field SEC.ACC.NO to CASH.BALANCE to give details of orders. This field contains a different value depending on whether this segment of the total order refers to a portfolio buying or selling securities. If this refers a portfolio buying a security then it is the total value of the portfolio in the security currency of the security being bought. If this refers to a portfolio selling a security then it is the current nominal holding of the security being sold by the portfolio. For TARGET order types this will be the current nominal holding.
