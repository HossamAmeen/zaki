      • getPartyid

        public TField getPartyid()
        Returns the Partyid

        Field description: This field gives the party for which the PTY Contract is defined. The PTY Contract party can be defined as a BIC in which case it can be specified as a BIC4, BIC6, BIC-8 or BIC-11. The Party ID type for a BIC is B. A Party ID Type and Party ID can therefore be defined for example as B BARC, B BARCGB, B BARCGB21 or B BARCGB2106P. (An Contract with a BIC-11 has a higher priority compared to an Contract defined at BIC-8. Similarly in terms of pririty - BIC-8 Contract > BIC-6 Contract > BIC-4 Contract) The Party ID can also be defined as a National Clearing Code (NCC). The Party ID type for a national clearing is N. A Party ID Type and Party ID can therefore be defined for example as - N SC200050

        Returns:
