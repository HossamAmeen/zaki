com.temenos.t24.api.hook.system

Class Delivery

Packaged in : DE.DeliveryHook.jar

The DeliveryHook component allows translate amounts into a particular language. and also allows to map additional data which is not available for the message type. There are hooks for ROUTINE in DE.MAPPING and FORMAT.ROUTINE in DE.WORDS.

        public String translateAmountInSpecificLanguage(TNumber amountToConvert, TNumber lineLength, TNumber numberOfLines)
        This Interface enables the implementer to perform the translation of amounts into a particular language.
        This is required where a language is not defined in DE.WORDS. This interface is invoked while running PRINT.OUT service.

        T24 details: Interface method for the FORMAT.ROUTINE field in DE.WORDS in T24. The EB.API hook used by this interface is HOOK.DE.WORDS.FORMAT.ROUTINE. If an exception is thrown in the implementing class exception message will be logged in Msg Err Cde field in DE.O.HEADER for the respective delivery reference Id.
        Parameters:
        amountToConvert - The amount to be converted.
        lineLength - Length of output line.
        numberOfLines - Number of output lines.
        Returns:
        String - s amounts in words from local development.
