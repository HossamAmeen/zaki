      • getRanking

        public TField getRanking()
        Returns the Ranking

        Field description: This fields specifies the rank of the SLA defined for a set of message priority, Code word, code word and tag. SLA for a payment will be determined for exact value present in the payment or the default value (*). When this results more than one SLAs then final SAL is determined based on the Ranking of the SLA. Lowest value in the Priority field will be considered as highest priority. The ranking of a SLA should be unique across the company. This is a mandatory fields and valid values are 1 to 999
