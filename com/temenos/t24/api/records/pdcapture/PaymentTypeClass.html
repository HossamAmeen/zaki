        public TField getSettlementAcct()
        Returns the SettlementAcct

        Field description: Identifies the account which is to be credited for the corresponding payment type and amount. The account must be in the same currency as that defined in the CURRENCY field. This field is part of an associated multivalue set. In case the underlying Contract Customer (eg. Must be of the same currency as defined in the CURRNECY field.
