      • addTakeover

        public void addTakeover(TField param)
        Appends the specified Takeover to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: This field will control how the data for a DW.EXPORT record will be extracted. This field has the following options: BULK, INCREMENTAL, DAILY, ONLINE

        BULK: This option will extract all the records.

        INCREMENTAL: This option will extract only those records that were updated or created newly.

        DAILY: This option is to be used only when a value is specified in the field EXTRACT.FILE. This will extract data from the file specified in the field EXTRACT.FILE

        ONLINE: It updates the target DW Online database in Near Real time. The multiple changes to the same record for same day will be consolidated as one record per day. ONLINE INTRA: It updates the target DW Online database in Near Real time. The Multiple Changes to the same record for same day will create multiple records for same day.

        Parameters:
        param - the Takeover to append
        Returns:
        nothing

        Field description: This field will control how the data for a DW.EXPORT record will be extracted. This field has the following options: BULK, INCREMENTAL, DAILY, ONLINE

        BULK: This option will extract all the records.

        INCREMENTAL: This option will extract only those records that were updated or created newly.

        DAILY: This option is to be used only when a value is specified in the field EXTRACT.FILE. This will extract data from the file specified in the field EXTRACT.FILE

        ONLINE: It updates the target DW Online database in Near Real time. The multiple changes to the same record for same day will be consolidated as one record per day. ONLINE INTRA: It updates the target DW Online database in Near Real time. The Multiple Changes to the same record for same day will create multiple records for same day.
