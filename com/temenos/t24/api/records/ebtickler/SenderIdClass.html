      • getSenderStat

        public TField getSenderStat()
        Returns the SenderStat

        Field description: Refers to the Tickler's status from the Sender's point of view. This defaults to 'SCHD', which means the Tickler is scheduled for User/s on a particular date. In those cases where more than on Receiver is set on a single date to carry out a tasks and the Sender intends to 'STOP' the job set for all the Receivers, this may be set to 'STOP', as a Top level measure. So when this is set to 'STOP' all the RCV.STATUS shall be set to 'STOP'. Validation Rules: Valid inputs are 'SCHD' and 'STOP'.
