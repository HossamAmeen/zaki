        public TField getAlBalAmtBef()
        Returns the AlBalAmtBef

        Field description: Holds the amount of the asset and liability CRB base in AL.CCY prior to conversion of the local currency, for the associated fields AL.CCY to CONT.LCL.AFT. This field is calculated by taking the BALANCE, DEBIT.MOVEMENT and CREDIT.MOVEMENT for each non-contingent asset type in each CONSOLIDATE.ASST.LIAB record for the AL.CCY.       • getAlPosAdjust

        public TField getAlPosAdjust()
        Returns the AlPosAdjust

        Field description: Contains the asset and liability currency position adjustment in the associated AL.CCY after the local currency is converted. This field forms part of the series of associated fields AL.CCY to CONT.LCL.AFT which hold control totals built when the local currency is converted. This field is calculated as the difference between AL.BAL.AMT.AFT and AL.POSN.AMT.AFT for the associated AL.CCY. This value is calculated immediately after the local currency conversion. This value will be raised when the associated AL.BAL.AMT.AFT if the AL.CCY in question is the ORIGINAL.LCY. Additional currency positions may need to raised to represent the effect of interest accruals and charges in the old currency which would not have raised a position. Validation Rules: This is a NOINPUT, system amount field.
