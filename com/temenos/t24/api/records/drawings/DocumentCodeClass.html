      • getDocumentCode

        public TField getDocumentCode()
        Returns the DocumentCode

        Field description: Used in the production of messages/advices. A list of the various types of documents that are commonly required under L/C's are detailed in the LC.ADVICE.TEXT file and recorded by an ID code. This field can therefore be used to display the documents which are being sent under cover of the documentary remittance letter.The full description of the document will be displayed in the field <DOCUMENT.TXT>. The field is associated with fields DOC.1ST.COPIES and DOC.2ND.COPIES below which indicate the post the documents will be sent in, and the number of copies. Validation Rules: 1-5 type A (Alphanumeric) characters. Must exist in LC.ADVICE.TEXT file. If documents required under the LC are not recorded in the LC ADVICE.TEXT file, they can be input separately in <ADDITIONAL.DOCS>.

        Parameters:
        Returns:
