      • getFinancialStatus

        public TField getFinancialStatus()
        Returns the FinancialStatus

        Field description: Provides a feature to specify different credit allocation based on arrangement's Financial Status i.e. based on arrangement being suspended or not. This will accept three options Both - Only one credit allocation irrespective of arrangement being suspended or Not. If "Both" selected then no other option can be specified. Performing - Specifies credit allocation to be applied on the arrangement when it is not suspended. Both is not alllowed and system would insist to define Suspend if not defined. Suspended - Specifies credit allocation to be applied on the arrangement when it is suspended. Both is not allowed and system would insist to define Performing if not defined.

        Parameters:
        Returns:
