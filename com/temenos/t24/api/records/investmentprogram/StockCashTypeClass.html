      • getStockCashType

        public TField getStockCashType()
        Returns the StockCashType

        Field description: This field is used to define the treatment of standing instructions for corporate actions, more details on this can be found in the Corporate Actions user guide. It is a Multi-valued field associated with all the field to STOCK.CASH. It will accept input of ALL, 'A-' followed by a valid ASSET.TYPE, 'S-' followed by a Valid SUB.ASSET.TYPE, 'L-' followed by a product type or a valid security number. Validation Rules: accept input of 'ALL', or 'A-' followed by a valid ASSET.TYPE, or 'S-' followed by a valid SUB.ASSET.TYPE, or 'L-' followed by a product type or Valid security number must exist in SECURITY.MASTER file.
