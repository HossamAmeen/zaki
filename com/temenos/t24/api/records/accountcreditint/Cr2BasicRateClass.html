        public TField getCr2BasicRate()
        Returns the Cr2BasicRate

        Field description: Field range CR2.BASIC.RATE to CR2.LIMIT.AMT are used to specify rates applicable to different balance limits. Specifies a Basic Rate which is held on the BASIC.INTEREST (Ref: General Tables) rate table. (CR2 fields specify a second interest which may be calculated). Rates can be fixed or linked to Basic Rates plus or minus a Margin. For examples see CR.BASIC.RATE. Validation Rules: 1 to 4 numeric characters. Multivalue field associated with field range CR2.BASIC.RATE to CR2.LIMIT.AMT. (Optional input. Not allowed if corresponding CR2.INT.RATE, is input or if INTEREST.DAY.BASIS contains NONE or GENERAL.) There must be a record in the BASIC.INTEREST table (Ref: General Tables) with a key consisting of this code, the Currency of the Account and an effective date which is less than or equal to the date specified in ID.

      • getCr2MarginOper

        public TField getCr2MarginOper()
        Returns the Cr2MarginOper

        Field description: Field range CR2.BASIC.RATE to CR2.LIMIT.AMT are used to specify rates applicable to different balance limits. Determines the method by which the Rate specified in corresponding fields CR2.BASIC.RATE or CR2.INT.RATE is adjusted by the Margin specified in corresponding field CR2.MARGIN.RATE. (CR2 fields specify a second interest which may be calculated.) If the Rate specified in CR2.BASIC.RATE or CR2.INT.RATE is to be adjusted by a Margin, this field specifies whether the Margin Rate is to be added to, subtracted from, or multiplied with the Basic Rate to arrive at the effective interest rate. If Multiply is specified the final rate is (100 + M) percent of the Rate from CR2.BASIC.RATE or CR2.INT.RATE, where M is the Margin Rate percentage from CR2.MARGIN.RATE. For examples see CR.BASIC.RATE. Validation Rules: ADD, SUBTRACT or MULTIPLY, or any abbreviation thereof. Multivalue field associated with field range CR2.BASIC.RATE to CR2.LIMIT.AMT. (Optional input. Only allowed if corresponding field CR2.BASIC.RATE or CR2.INT.RATE is input.)

        Parameters:
        Returns:
        TField - The Cr2MarginOper
