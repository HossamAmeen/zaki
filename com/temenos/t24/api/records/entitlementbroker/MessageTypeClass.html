      • getMessageType

        public TField getMessageType()
        Returns the MessageType

        Field description: Identifies the delivery message types generated for this record. The actual message types are defined on EB.ADVICES. Currently the following specific "hard coded" confirmation message types are supported by Corporate Actions: 1505 - Dividend Tax Certificate 1510 - Capital Increase Advice 1520 - New Issue Advice 1530 - Coupon /Dividend Advice 1540 - Stock Dividend Advice 1560 - Redemption Advice Additionally if new message types are required SOFT DELIVERY processing will be invoked as described in the DELIVERY section of the T24 user guide. Information handed to delivery is always comprised of 9 records, see the Delivery user guide for more information. The information handed off to delivery for new message types will be as follows Record 1 equates to data defined in DE.MESSAGE 1500, Corporate action generic message Record 2 comprises the Entitlement record. Record 3 comprises the Diary record. Record 4 comprises the Security Account Master record. Record 5 comprises the Security Master record. Records6 to 9 are left blank. If a user routine is defined in USER.ROUTINE of the relevant EB.ADVICES record then this routine will be invoked and used to create Record9. Validation Rules: No Input field.

        Parameters:
        Returns:
        TField - The MessageType
