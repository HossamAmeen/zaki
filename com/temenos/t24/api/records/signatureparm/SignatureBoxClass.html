        public TField getBottomRight()
        Returns the BottomRight

        Field description: Defines the screen co-ordinates for the bottom right corner of a signature box. Each signature box is positioned on the screen by specifying the TOP.LEFT and BOTTOM.RIGHT screen co-ordinates. This field is multivalue associated with SIGNATURE.BOX. Validation Rules: 1 - 3 numeric ',' 1 -3 numeric Multivalue associated with fields 7 to 12 Enter as XXX,YYY where XXX = X axis co-ordinate in the range 1-1000 YYY = Y axis co-ordinate in the range 026-750 Co-ordinates are converted at display time into a fraction of the available screen area of the terminal being used for display. The 026 minimum on the Y axis allows for the status line on the signature card.

      • getDisplayName

        public TField getDisplayName()
        Returns the DisplayName

        Field description: Defines whether any name saved with a signature is to displayed on the label line of the signature box. When signatures are captured or at a later stage if required a name may be added for each signature box. If this field is set to 'Y' for any signature box then the name if it exists will be appended to the actual signature box label and displayed at the screen position defined in LABEL.POSN. This field is multivalue associated with SIGNATURE.BOX. The default setting is 'N'. Validation Rules: 1 alphanumeric Multivalue associated with fields 7 to 12 Input must be 'Y' or 'N' only. Defaults to 'N'
