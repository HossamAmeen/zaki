      • getAcctType

        public TField getAcctType()
        Returns the AcctType

        Field description: This field is used to indicate what role the terminal/machine plays for the processing code. The terminal ID might be the same but processing can be done both for cash withdrawal and POS transactions. So this field will indicate the role or the type of transaction attached to the processing code. The value for the field is provided in the lookup table ATM.PROC.ACCT.TYPE. The existing set of records are provided in the list available in Data records section. The field should be used appropriately to define the Business.For e.g. Same BIN no cannot be used to act for ON US and OFF US Transactinos.
