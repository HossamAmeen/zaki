      • getAddressAttribute

        public TField getAddressAttribute()
        Returns the AddressAttribute

        Field description: Identifies a name or address attribute that will be included in the output item. It is part of the output item group. Adding multiple address attributes within the same output item, implies that all the address attributes will be combined together within the same output item (it will apply the AND logic) in the order specified by user. Attributes that are combined together are separated by a space. This is a multi-value mandatory field. Validation Rules: The value inputted in this field will be validated against the fields from DE.ADDRESS. No duplicate allowed per CarrierType.
