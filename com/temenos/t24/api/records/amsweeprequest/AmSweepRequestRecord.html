      • getMinCashAmt

        public TField getMinCashAmt()
        Returns the MinCashAmt

        Field description: This field holds the required target cash value that the traded currency account should be swept to by the transfer of funds between the traded currency account and the counter currency account within a portfolio. The operation type field determines whether the value of the traded currency account should be reduced or increased to this value. In the case of a Sell operation, only accounts in excess of this value are processed with the excess being swept into the counter currency account. In the case of a Buy operation, only accounts short of this value are processed with the shortfall being swept from the counter currency account. This amount is expressed in system base currency

        Returns:
