com.temenos.t24.api.hook.contract

Class Calculation

Packaged in : ST.CalculationHook.jar

The CalculationHook component allows access to routines for customising the rates, tax and sweep amount etc. Customizing the values are through the exit points of T24 files like TAX, PERIODIC.INTEREST, FT.COMMISSION.TYPE and AC.SWEEP.TYPE.

        public void calculateBaseAmount(String taxId, TDate startDate, TDate endDate, String applicationName, String recordId, String localCurrency, TBoolean runningInCob, TStructure record, TNumber baseAmount)
        This interface enables the implementer to calculate the Base amount during the tax calculation process, literally it is the deal amount passing during charge calculation, tax splitting process etc. This interface is invoked in online process like posting FT, close of Business or at the time of LD, MM commitment etc. With the return base amount,it will modify the deal amount when calculating the charges or spliting tax process etc. T24 Details: Interface method for the BASE.AMT.ROUTINE field in TAX in T24. If an exception is thrown in the implementing class during online process it gets displayed in the browser with the prefix "EB-EXCEPTION.RAISED". It it is during COB, then it logs in T24 file "EXCEPTION.LOG.FILE"
        Parameters:
        taxId - Id of the tax to which the base amount is to be calculated
        startDate - The start date for the tax calculation
        endDate - The end date for the tax calculation
        applicationName - Name of the application from where this interface is being triggered
        recordId - The id of the record through whcih this interface is being triggered
        localCurrency - The currecnt company's local currency
        runningInCob - Denotes for running in close of business or not
        record - The record data appropriate to the 'recordId' parameter
        baseAmount - The base amount has been pass as in param and returned the new updated value
        Returns:
        nothing
