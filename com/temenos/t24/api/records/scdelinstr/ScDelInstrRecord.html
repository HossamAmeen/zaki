        public TField getAutoSettleLimit()

        Field description: Indicates whether automatic settlement of counterparty limits take place. Input to this field is only valid if both TO.BE.RECD.OR.DEL and COUNTERPARTY.LIMIT are set to Y. If automatic settlement of counterparty limits take place then during the batch relevant to the value date of the SEC.TRADE deal the corresponding SEC.DEL.CONTROL record will updated as if complete settlement has taken place and the counterparty limit will reflect this. If automatic settlement is set to NO (or left blank) then the user must update the relevant SEC.DEL.CONTROL record to indicate that delivery has taken place at which stage limits will be updated to reflect this. Validation Rules: Optional input. Input is only allowed if TO.BE.RECD.OR.DEL and COUNTERPARTY.LIMIT are set to Y. If input is made must be either Y or NO.
