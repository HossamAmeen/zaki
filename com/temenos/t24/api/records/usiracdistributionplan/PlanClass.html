      • getDistributionSeq

        public TField getDistributionSeq()
        Returns the DistributionSeq

        Field description: Defines the IRA accounts that funds the calculated RMD. This field provides a drop down that contains the following values: Lowest Account Balance - sequentially uses the IRA account, based on lowest balance first, within the designated plan to fund the RMD with exhausted accounts moving through the closing process. Lowest Interest Rate - sequentially uses the IRA account, based on lowest rate first, within the designated plan to fund RMD with exhausted accounts moving through the closing process. Earliest Maturity - sequentially uses the IRA account, based on earliest maturity date first, within the designated plan to fund the RMD with exhausted accounts moving through the closing process. Manual - provides the bank the ability to selected a client-defined, funding-priority of funding.
