      • getReschedNotice

        public TField getReschedNotice()
        Returns the ReschedNotice

        Field description: Defines the notice period in days for the corresponding reschedule type, for all the AZ accounts opened under this product. For a particular type of reschedule, the reschedule notice will be linked and the reschedule event will happen after the expiry of reschedule notice. The valid types of reschedules are: 1. Rate Change 2. Lump sum Repayment 3. Redraw 4. Term Change 5. Frequency Change Note: At present, only Rate Change is supported. If the BASIC.INTEREST is changed today or for a forward date after the contract is input then this change is applied to the az.account after the number of days defined in this field. The BI change is processed during COB. Validation Rules: Multi value field associated with Reschedule Type Input allowed only if reschedule Type is defined 1-99 numeric character with period D, M or Y (Example: 15D, 01M) Not applicable for credit card loans
