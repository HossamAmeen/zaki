      • getRoBaseCcyAmt

        public TField getRoBaseCcyAmt()
        Returns the RoBaseCcyAmt

        Field description: This field indicates the base/facility currency equivalent of the amount of drawing being considered for ROLLOVER, MERGER or SPLIT operation. If a drawing in a currency other than the base currency is selected, then the base currency equivalent of the drawing as updated at the time of input of the drawing is updated in this field. If a value is entered in the Prn Dec field, the net amount being considered for ROLLOVER, MERGER or SPLIT operation is converted into base currency equivalent. The exchange rate used for conversion is the same as the rate used for conversion when the drawing was input. When a drawing in base currency is selected the drawing amount is written in this field. Validation Rules: System Written No input allowed

        Parameters:
        Returns:
        TField - The RoBaseCcyAmt
