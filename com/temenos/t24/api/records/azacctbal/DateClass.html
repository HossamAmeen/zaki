      • getIntAdjAmt

        public TField getIntAdjAmt()
        Returns the IntAdjAmt

        Field description: This field holds the interest amount which is adjusted from the suspense account due to the interest payment amount lesser than the accrued interest amount. In ANNUITY or CREDIT-CARD type loan accounts, it is possible to make an interest payment less than the accrued interest. In this case, suspense account from the class AZINTADJ would be debited for the remaining amount and this field would get updated. This suspense account would be settled during subsequent payments. This field is used only if LOAN.INT.ADJUST field in AZ.PRODUCT.PARAMETER record is set to YEs

        Returns:

        public TField getIraAmount()
        Returns the IraAmount

        Field description: This field holds the interest amount which is received in advance. In an AZ loan contract, when an interest payment is made, the excess amount received will be parked in a sub account and this field would get updated with the excess amount. During next interest payment, the sub account would be debited and this field would be updated based on the remaining amount in the sub account. This field is used only if PROCESS.IRA field in AZ.PRODUCT.PARAMETER application is set to YES. .
