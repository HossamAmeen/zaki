      • getIdlAdvisedAmount

        public TField getIdlAdvisedAmount()
        Returns the IdlAdvisedAmount

        Field description: The advised amount of the intraday limit. This is the IDL amount which has actually been advised to the customer and will be added to the available amount for credit checking, during the agreed IDL time period during the day only. The amount recorded in this field is an extension to the agreed customer category limit. In order to make the intraday limit unavailable immediately the intraday limit advised amount can be set to zero. Validation Rules: Standard T24 amount. Must have a value in line with IDL.EFFECTIVE.DATE.
