        public TField getLocalBondsTax()
        Returns the LocalBondsTax

        Field description: This field should be used to record the tax code to be applied when calculating the LOCAL tax on coupons in respect of bond type securities. It should contain either a valid tax code identifying a specific tax record from the TAX file or the key of a specific record on the TAX.TYPE.CONDITION file. In the case of the latter, the id must always be prefixed by a " * ". For example " *USWHT ". Note - a bond type security is identified by having the BOND.OR.SHARE field on its underlying SECURITY.MASTER file set to "B". Validation Rules: Mandatory field Any value input must exist on either TAX or TAX.TYPE.CONDITION files If TAX.TYPE.CONDITION record required, then must be prefixed with a " * "

        public TField getLocalTaxCust()
        Returns the LocalTaxCust

        Field description: This field should be used to specify whether, when calculating LOCAL tax using the rules that may have been defined within the TAX.TYPE.CONDITION file, the conditions applicable to the customer or the depository are to be used. For example, if the tax being deducted at source is dependent upon simple residency, then you will need to specify whether it is the residency of the depository or the residency of the customer(s) to whom the dividend or coupon is to be paid.
