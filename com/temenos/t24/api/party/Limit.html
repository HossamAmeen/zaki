Class Limit

Packaged in : LI.LimitApi.jar

Provides methods to get the balances for a given Limit. The member variable for this class is limitId, which must be set for correct use of the methods.

      • getCurrencyAmount

        public LimitCurrencyAmount getCurrencyAmount(TBoolean checkValueDate, TDate transactionDate)
        Returns a list of amounts calculated for the accounts from the LIMIT table for the given limitId, and also calculates a consolidated LimitAmount in the LIMIT.CURRENCY of the given limit.
        T24 Details: This invokes the T24 routine LIMIT.GET.ACC.BALS.
        Parameters:
        checkValueDate - Optional. set to TRUE to consider the value dated balance. Default is FALSE.
        transactionDate - Required. Can be set to past dates to return historical data.
        Returns:
        LimitCurrencyAmount - the LimitAmount
        Throws:
        exception for null limitId and core exception
