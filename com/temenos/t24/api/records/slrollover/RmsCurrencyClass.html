      • getExchVariance

        public TField getExchVariance()
        Returns the ExchVariance

        Field description: Applicable when an existing drawing denominated in one of the optional currency (any allowed currency other than the base currency) is rolled over into the same optional currency. If the RMS amount (new deal amount) exceeds the original drawing amount by a percentage greater than the value mentioned in this field, then RMS amount is retained as the new drawing amount. If the variance is less than the value in this field the old drawing amount is retained as the RMS amount. During Rollover if any Increase or Decrease of Principal if effected, then this variance will not be checked and the Transactions will occur at current rate only. Validation Rules: Default from facility file. Applicable for rollover operation only. Applicable when a drawing in an optional currency is rolled over in the same optional currency.

        Parameters:
        Returns:
        TField - The ExchVariance
