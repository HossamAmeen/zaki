        public TField getDeProcess()
        Returns the DeProcess

        Field description: Indicates the Delivery Process that is to be carried out for the respective message after Condition Satisfies. If the value is HOLD then the Delivery Process is held and the details are written to TI.TRANS.DETAILS. If the value is DELETE no delivery process occurs. Normal Delivery process happens if the value of this field is ONLINE If not input and Message Number exist then Defaulted to 'ONLINE'. Validation Rules: Input to this field is optional. The valid input to this field are "HOLD' , 'DELETE' or 'ONLINE'. If not input and if corresponding Message Number exist then Defaulted to 'ONLINE'.
