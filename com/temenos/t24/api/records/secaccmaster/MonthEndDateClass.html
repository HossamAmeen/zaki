      • getAdvisoryFees

        public TField getAdvisoryFees()
        Returns the AdvisoryFees

        Field description: Multi-valued no input field associated with the MONTH.END.DATE and SAFECUSTDY.FEES fields to form a multi-valuedset of fields containing the fees due from the portfolio but not yet taken. Where SAFECUSTODY.VALUES>PERFORM.ACCRUAL for Charge.Type 'IC' is set to 'MONTHLY', this field will containthe Management Fees due for each of the months in the associated MONTH.END.DATE field. This field will be cleareddown when the fees are taken and the SC.ADVISORY.CHG record for the portfolio is updated with them. Only updated for customer portfolios where the PORTFOLIO.FEES field is AUTOMATIC. Not updated by DAILY accrual Validation Rules: No input field, automatically updated by the system.
