      • getInternalFreq

        public TField getInternalFreq()
        Returns the InternalFreq

        Field description: Specifies the frequency a particular report is to be produced for internal purposes. The type of reports are detailed in INTERNAL.REPS. Note : This is a default frequency it is to be used in conjunction with the multi valued INTERNAL.REPS field to define the various reports required by a particular client, and the frequency of those reports. By defining internal and external reports the user is able to print all external reports on high quality statement paper (possibly pre-printed for this precise purpose). Different printers can be attributed to these reports to avoid unnecessary paper and time wastage. Validation Rules: Up to 17 alpha characters available Must exist as a valid frequency on the frequency table Multi-value set with INTERNAL.REPS
