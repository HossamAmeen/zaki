      • getScreenHdr0139

        public TField getScreenHdr0139()
        Returns the ScreenHdr0139

        Field description: Enables the headings for columns 1-39 of the Screen to be specified. his field can be used to specify the column headings for use with the LIST Function. Spaces will not be cancelled. It is important to note that the System will always reserve the three first columns of the Data Section to display the line number of each record. This line number will be followed by a space and the ID of the record. The number of columns reserved for the ID will be in accordance with the ID of each Application (e.g. 6 for CUSTOMER file, 4 for SECTOR code etc...). The User should thus start in this field with a Text like 'SEQ REF ....' and then add his own header text. Validation Rules: 3-39 type A (alphanumeric) characters including required spaces. (Mandatory input. No default value.)

        Parameters:
        Returns:
