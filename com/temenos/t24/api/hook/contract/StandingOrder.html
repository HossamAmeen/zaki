Class StandingOrder

        public void checkStandingOrderFundsTransfer(String ftReferenceId, String ftRecordStatus)
        This Interface enables the implementer to check the status of a Funds Transfer generated by a standing order during batch processing.
        An implementor would typically record failed transactions in IHLD status for later reprocessing.
        This interface is invoked after the FT generated by the Standing Order has been processed or resubmitted in the Batch.
