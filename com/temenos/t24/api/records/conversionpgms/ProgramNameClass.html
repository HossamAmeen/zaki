      • getProgramLevel

        public TField getProgramLevel()
        Returns the ProgramLevel

        Field description: Defines the level at which the conversion program should run. When this field is entered, the sub-value set from companies selected to run information is expanded with the company code of all the companies in which the conversion program should be run, depending on the level and application of the conversion program. For example, if a securities conversion program is defined as being of FINancial level, the company file will be accessed to determine all financial non-consolidation, non-reporting companies in which securities is installed. The sub-valued set will then be expanded to contain these companies, with the run pgm flag set to 1, i.e. the program is to be run. If there are no companies in which the conversion program is to be run (e.g. securities has not been installed in any company), the sub-value set will be null. Validation Rules: Values are: INT (Installation) CUS (Customer) FIN (Financial) FTF (Financial table - financial information) FTD (Financial table - non-financial) CCY (Currency) NOS (Nostro) CST (Customer table) PROGRAM.NAME through to RUN.INFO form a multivalue set describing each conversion program to be run. Optional input. Default value INT.

        Parameters:
        Returns:
        TField - The ProgramLevel
