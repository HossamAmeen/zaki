        Field description: Specifies the value (or values) for the associated SELECTION.FIELD and OPERAND which must be true in order to request a contract or account conversion.       • getSelectField

        public TField getSelectField()
        Returns the SelectField

        Field description: Specifies the name of the field defined in the STANDARD.SELECTION record for the associated APPLICATION to be used to perform the selection specified in the associated fields OPERAND and VALUE. The field used may be of type 'D', 'I', 'J'. Multiple field selections can be specified by adding additional values to this field. Where multiple selection fields are used the combination is treated as 'AND' when performing the selection, so that records are only selected where each selection conidtion is true. Validation Rules: At least one selection is required. Must be a field of type 'I', 'D' or 'J' defined in the STANDARD.SELECTION record for the APPLICATION.
