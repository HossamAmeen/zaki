        Field description: Specifies if the exception amounts held in the associated AL.CCY.EXCEP and AL.LCY.EXCEP are to be automatically adjusted so that the position total matches the CRB total. If set to Y the system will adjust the values and the adjustment amount raised will be held in the AL.POS.ADJUST and AL.POS.LCY.ADJ fields.
