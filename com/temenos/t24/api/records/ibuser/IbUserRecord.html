        Field description: Specifies the separators to be used when formatting amounts for enquiry and record display in T24.This field specifies the separators to be used when formatting amounts. Input of ',.' will use commas to separate the thousands and a full stop for the decimal place.Input of '.,' will have the opposite effect.Any combination of 2 from comma, full stop and apostrophe can be used.Data returned to the external system formatted by T24 (e.g. from a typical enquiry) will be formatted according to this option, the display of data by the external system using this format will depend upon the operation of that system. Validation Rules (Optional input)Must be either comma, full stop or apostophe.The same character cannot be used twice.The default is ',.'.
