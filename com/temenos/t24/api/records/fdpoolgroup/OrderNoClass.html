        public TField getOrderStatus()
        Returns the OrderStatus

        Field description: The status of the related FD.FID.ORDER contract referenced in the related field ORDER.NO is held in this field. The following values will be used: OPEN ==== The order is currently open and awaiting approval by the dealer. APPROVED ======== The order has been approved by the dealer. EXIST APPROVED ============== When an order has this status, the order has been added to this pool group after the placement has been created and the order has been approved by the dealer. However, only when the order is approved on the fiduciary placement will the status of the order be set to APPROVED. Fields ORDER.NO to DEALER.NOTES form a related multi value set.
