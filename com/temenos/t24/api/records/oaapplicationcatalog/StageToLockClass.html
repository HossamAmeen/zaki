      • getStageRule

        public TField getStageRule()
        Returns the StageRule

        Field description: Entry in this field should be a valid record from OA.RULE.DEFINITION, wherein a Boolean rule can be specified. This field indicates the rule that, if evaluated as TRUE, will cause fields specified in STAGE.NOINPUT.FIELD or STAGE.INPUT.FIELD to be exempted from the STAGE.DEFAULT setting Example: if STAGE.DEFAULT is NOINPUT , input (or change) of values in fields specified in STAGE.INPUT.FIELD will be allowed only if the STAGE.RULE evaluates as TRUE If STAGE.RULE is specified, STAGE.NOINPUT.FLD.RULE or STAGE.INPUT.FLD.RULE cannot be specified

        Input to this field should be a RULE expressed in the format {FIELD.NAME}{COMPARISON.OPERATOR} {COMPARISON.VALUE} FIELD.NAME should be a valid field for the standard selection of the table where the rule is evaluated.Example: OA.APPLICATION ,OA.APP.FORMLET.CLASS Multi- value part of a set starting with STAGE.TO.LOCK, including STAGE.DEFAULT , STAGE.RULE, STAGE.NOINPUT.FIELD, STAGE.NOINPUT.FLD.RULE, STAGE.INPUT.FIELD and STAGE.INPUT.FLD.RULE Optional Input

        Parameters:
        Returns:
        TField - The StageRule
