      • addPriorityX

        public void addPriorityX(TField param)
        Appends the specified PriorityX to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Defines the priority of X-axis members. Validations:

        Mandatory input field, when the system is prioritised to do rebalancing. This is based on the matrix priority defined in X axis.

        Duplicates are not allowed.

        No additional members can be defined in the priority fields, i.e. only those members which are defined in the matrix can have a priority.

        Does not accept any other members in these fields.

        All the members which are defined along the X axis must have a priority, if the system is prioritised for rebalancing based on matrix priority.

        Parameters:
        param - the PriorityX to append
        Returns:
        nothing
