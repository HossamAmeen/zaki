        public void addOwnRepStatus(TField param)
        Appends the specified OwnRepStatus to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: The statuses (multi-value) for which the owner details beneficial and substantial - need to be reported. For the accounts where the owner details are to be reported based on the status (classifications) defined here, the system will extract the details of beneficial and substantial owners (as applicable) and update the base file. The beneficial and substantial owner roles will be identified based on roles defined in FATCA ROLE TYPE. As per the Form 8966 reporting, this will be required for OD FFIs and Passive NFFEs with identified US owners. Validation Rules: A valid FATCA TAX STATUS that has been defined as a REPORTABLE STATUS.
