Class InternationalAccountingStandards

Packaged in : IA.AccountingHook.jar

        public void calculateContractBalance(String calculatedAmountType, List<TNumber> referenceRateIds, IasApplicationParamRecord iasApplicationParamRecord, String contractApplication, String contractId, TStructure contractRecord, String baseApplication, TStructure baseRecord, IasContractBalancesRecord iasContractBalancesRecord, String productGroupId, String action, TString contractBalance, TString contractLocalCurrencyBalance)
        This Interface enables the implementer to calculate the latest balance and latest balance in local currency with respect to the amount type. This interface is invoked at both stage online/COB.
        T24 details:Interface method for the DEF.EXTRACT.RTN field and DEF.VALUE.RTN in IAS.AMOUNT.TYPE.
        Its also used for BUILD.AMT.RTN and MODIFY.AMT.RTN in IAS.APPLICATION.PARAM. The EB.API hook used by this interface is HOOK.IAS.APPL.PARAM.BUILD.AMT.RTN.         If exception is thrown in the implementing class exception message will be logged in EB.EXCEPTION.LOG.
        Parameters:
        calculatedAmountType - Calculated Amount type.
        referenceRateIds - Ids of the Reference Rate Table which is specified REL.RATE.TYPE field in IAS.AMOUNT.TYPE.
        iasApplicationParamRecord - Record of the IAS.APPLICATION.PARAM application.
        contractApplication - Name of the contract application. EX: Any valid T24 application Name.
        contractId - Id of the main CONTRACT which has been Input.
        contractRecord - Record of the contract which has been Input.
        baseApplication - BASE application specified in the IAS.APPLICATION.PARAM for the application.
        baseRecord - Record of the BASE Application.
        iasContractBalancesRecord - IAS.CONTRACT.BALANCES record which will already be updated with the static Info and the RATE.TYPES.
        productGroupId - Id of the IAS.PRODUCT.GROUP applicable for the contract.
        action - ACTION can be BUILD or MODIFY.
        contractBalance - Contract's latest balance.
        contractLocalCurrencyBalance - Contract's local currency equivalent to the latest balance.
        Returns:
        nothing
