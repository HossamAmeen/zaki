      • getCommodity

        public TField getCommodity()
        Returns the Commodity

        Field description: Associates a Limit with a specific Industry or Industries. Where more than one industry is specified, a percentage of the risk has to be defined in the Commodity Percentage field. This information is used to produce the Industry exposure report. If the Customers, within a Liability Group, are involved in a variety of different industries, then multiple entries of this sub-value field can be used to record this. The associated field COMMODITY PCENT is used to specify the percentage involvement in each of the Industries. Note: The commodity (Industry) exposure report will detail the utilisation (outstandings) according to the industries and percentages defined in the associated fields COMMODITY and COMMODITY PCENT (Fields 56.1 and 57.1). Validation Rules: 1-10 Numeric character Industry Code. (Optional input. Default to the Industry Code on the Customer record of the Liability Number.) Only for Customer level Group Limits (For e.g. M0000000020.2000.01.10020) this field will be considered.

        Parameters:
        Returns:
