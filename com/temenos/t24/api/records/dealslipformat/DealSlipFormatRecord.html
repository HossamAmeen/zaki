      • addDescription

        public void addDescription(TField param)
        Appends the specified Description to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Multi-language description of the deal slip format record. The REPORT.CONTROL id to be used if the deal slip is spooled rather than printed directly from the terminal. If the terminal does not support an attached printer, i.e. the LOCAL.PRINT fields are not defined in the TERMINAL record, then the deal slip will be spooled via REPORT.CONTROL. This field is the REPORT.CONTROL key to use and will ultimately define the report environment, destination printer, attributes etc. If this field is left blank and the deal slip is spooled then the key "P.FUNCTION" is used. Validation Rules: 3-35 type A (alphanumeric) characters.

        Parameters:
        param - the Description to append
        Returns:
