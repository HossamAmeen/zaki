      • calculateStampTaxAndEbvFees

        public void calculateStampTaxAndEbvFees(String applicationName, TBoolean isRunningInCob, SecurityValuationData securityValuationData, UserContext userContext, TaxData taxData, StampTaxData stampTaxData, EbvFeesData ebvFeesData)
        This interface enables the implementer to calculate the stamp tax and EBV(Effectenboursenverein) fees amount during the tax calculation process, literally it is the deal amount passing during tax calculation etc.
        This interface is invoked in online process like posting security transfer, sec trade and also during corporation actions service.
        which specifies the country specific used to calculate stamp tax and/or cantonal tax and EBV(Effectenboursenverein) fees where applicable.

        T24 Details: The EB.API hook used by this interface is SC.PARAMETER.TAX.SUBROUTINE.HOOK. The T24 field specifying this hook is the TAX.SUBROUTINE field in SC.PARAMETER.
        If an exception is thrown in the implementing class, the calculation will be cancelled; if running within a service the error will additionally be stored to the EXCEPTION.LOG.
        Parameters:
        applicationName - Name of the application from where this interface is being triggered
        isRunningInCob - Denotes for running in close of business or not
        securityValuationData - Denotes the security valuation data to calculate stamp tax and ebv fees
        userContext - Denotes the data to get the associated categories from the LOCAL6 common variable in I_COMMON
        taxData - Denotes tax data necessary for calculation of tax
        stampTaxData - Denotes the stamp tax calculation
        ebvFeesData - Denotes the ebv(Effectenboursenverein) fees calculation
        Returns:
        nothing

        public List<String> getSecurityPortfolioContracts(String applicationName, TBoolean isRunningInCob, String securityAccountMasterId)
        This interface enables the implementer to get the orders or contracts for the corresponding security account portfolio.
        This interface is invoked while running the enquiry SC.VAL.COST.
        The security account are passed in to get the respective orders or contracts.

        T24 Details: The EB.API hook used by this interface is SC.PARAMETER.USR.POR.SEL.RTN.HOOK. The T24 field specifying this hook is the USR.POR.SEL.RTN field in SC.PARAMETER.
        The associated field that specifies the valuation application for this method is VAL.APPL in SC.PARAMETER.
        If an exception is thrown in the implementing class, the enquiry will be cancelled; if running within a service the error will additionally be stored to the EXCEPTION.LOG.
        Parameters:
        applicationName - Name of the application from where this interface is being triggered
        isRunningInCob - Denotes for running in close of business or not
        securityAccountMasterId - The portfolio for which the contracts are to be fetched
        Returns:
        List<String> - A list of contractids for the associated asset defined in the SC.PARAMETER table.
