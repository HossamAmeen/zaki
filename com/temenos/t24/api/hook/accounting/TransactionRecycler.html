        public void modifyRecyclingCaptureData(TStructure incomingRecord, String aaArrangementId, TDate aaActivityEffectiveDate, TDate todayDate, AaAccountDetailsRecord aaAccountDetails, RcDetailRecord rcDetailRecord, TString retError)
        This interface enables the implementer to modify RC.DETAIL record. This interface is invoked during the recycling process in the COB. This field is of YES/NO type, if it set as 'YES', then the system checks for the local hook, otherwise it wont. T24 Details: Interface method for the LOC.CAPTURE.RTN field of RC.TYPE file. The hook name must be in the form of "RC.CAPTURE".RC.TYPE.ID."LOC". If an exception is thrown in the implementing class, the exception message will be catalogged in the EXCEPTION.LOG.FILE in T24, since it is running in COB.
        Parameters:
        incomingRecord - Record in STMT.ENTRY format, during COB through EOD.ACCOUNTING or ST.ContractHook:HandOffFormat format, during online process
        aaArrangementId - An account's arrangementId , eg 'AA16082SXNN3'
        aaActivityEffectiveDate - Activity Effective Date , eg '20170523'
        todayDate - Today's T24 date e.g '20170420'
        aaAccountDetails - It is the record of the AA.ACCOUNT.DETAILS
        rcDetailRecord - It is the record of the RC.DETAIL
        retError - Exceptions are thrown though this parameter
        Returns:
        nothing

        public void setTransactionCapture(StmtEntryRecord entryRrecord, HandOffFormat handOffSettelmentData, TBoolean setCapture)
        This interface enables the implementer to decide locally, if capture of a transaction is allowed or not for recycling process. This interface is invoked during the recycling process in the COB. This field is of YES/NO type, if it set as 'YES', then the system checks for the local hook, otherwise it wont. T24 Details: Interface method for the LOC.CAP.CHK.RTN field in RC.TYPE file in T24. The hook name must be in the form of "RC.CAPTURE.CHK.":RC.TYPE.ID:".LOC". If an exception is thrown in the implementing class, the exception message will be catalogged in the EXCEPTION.LOG.FILEin T24, since it is running in COB.
        Parameters:
        entryRrecord - Entry record in STMT.ENTRY format
        handOffSettelmentData - HandOff record in RC.ContractHook:HandOffFormat format
        setCapture - YES - capture allowed, otherwise not allowed
        Returns:
        nothing
