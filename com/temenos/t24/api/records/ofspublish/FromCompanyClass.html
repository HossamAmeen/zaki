        public TField getSelectFldName()
        Returns the SelectFldName

        Field description: Indicates the name of the field in the APPLN.NAME to be tested or a subroutine to test for publication. When it is the name of a field, it is used together with Fields DATA.COMPARISON to TO.DATA.It specifies precisely which records may be published from the application specified in the corresponding APPLN.NAME.A record may only be published if the field indicated here has a value allowed in the corresponding Fields DATA.COMPARISON to TO.DATA. To use @ID, the field name corresponding to the @ID should be used.For example for the application, PERIODIC.INTEREST, REF.CCY.DATE must be used instead of @ID. If a subroutine is used, routine name must start with @, the text after the @ must be a valid record on the PGM.FILE.
