      • getPropApplType

        public TField getPropApplType()
        Returns the PropApplType

        Field description:

        This field represents the balance to be paid for the defined property. The values allowed at present are BALANCES and NULL. If PROP.APPL.TYPE is null, the due balance in the bill for the property is settled according to the type defined in the field Application.Type. If PROP.APPL.TYPE is current, the current balance of the property is settled. (e.g. current principal, accrued interest, etc). This field allows us to settle current balance along with the due balances. If the field APPLICATION.TYPE has a value CURRENT, value in the field PROP.APPL.TYPE should be BALANCES for all the properties defined. This field forms a associated multivalue set with property and sequence

        Parameters:
        Returns:
        TField - The PropApplType
