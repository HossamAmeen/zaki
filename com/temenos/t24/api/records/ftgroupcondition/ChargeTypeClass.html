      • getChargePercent

        public TField getChargePercent()
        Returns the ChargePercent

        Field description: Specifies the rate used to calculate the Charge amount as a percentage of the standard Charge amount. Where groups of Customers are given preferential charge rates this field is used to define what percentage of standard charges (as defined in Charge Type table) should be applied. Where no charge is to be applied rate must be input as 0%. Validation Rules: Up to 10 type R characters plus a decimal point (standard rate format). Associated multivalue field. (Mandatory input if any Charge Types are input and Charge Currency is blank, otherwise not allowed.) Not more than 3 integers may be entered. Not more than 9 decimals may be entered.

        Parameters:
        Returns:
        TField - The ChargePercent

      • getChargeType

        public TField getChargeType()
        Returns the ChargeType

        Field description: Defines the Charge Types for which the Group of Customers will be afforded special conditions. Where a group of Customers are afforded special charge rates this field defines the Charge Types to which special rates apply. The Charge Type code entered in this field must first be defined in the Charge Type table. The 'ALL' condition will indicate that the percentage indicated in Charge Percentage is applicable to all charges for the selection. For any Charge Type not defined in this field, the Funds Transfer Application will automatically take the normal conditions as for the other Charge Types which are included in the Transaction Type table. Validation Rules: 2 type A (alphanumeric) character Charge Type. or ALL Associated multivalue field. (Optional input.)

        Parameters:
        Returns:
        TField - The ChargeType
