        public TField getAdjIntAmt()
        Returns the AdjIntAmt

        Field description: This field will hold the adjusted interest amount after the correction process. This field will be updated only when the defered tax amount is posted. The information in this field will be updated based on the file IC.CORRECTION.DETAILS. Once updated, this information will be cleared from the file IC.CORRECTION.DETAILS. If this field contains the value then it indicates that the previous period correction tax amount is posted on the current period. (this period)
