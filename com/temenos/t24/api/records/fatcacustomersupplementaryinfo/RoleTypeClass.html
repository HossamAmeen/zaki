      • getHolderRef

        public TField getHolderRef()
        Returns the HolderRef

        Field description:

        The field is used to specify a unique ID for the joint, beneficial or substantial owner. The Unique ID is then used in the FORM OWNER field to record the documents received from other owners (joint and beneficial).

        Validation rules: The value should contains two parts, first part should be FATCA CUSTOMER.SUPPLEMENTARY INFO ID, separator hyphen -, second part with sequence number. For example, if the record is being input for customer 100291, the HOLDER REF will be 100291-1 for the first beneficial owner
