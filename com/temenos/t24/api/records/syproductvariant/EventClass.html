      • getCobPhase

        public TField getCobPhase()
        Returns the CobPhase

        Field description: Different SY.PRODUCTS and variants normally follow common SY.EVENTs like Accrual, Fixing etc. The commonSY.EVENT.DEFINITION record can be set to either run ONLINE or during COB. However, for certain specific SY Productvariants, a particular activity may need to run Online rather than during COB and vice-versa. This value in thisfield takes presendence over value in COB phase of SY.EVENT.DEFINITION. For example,in the SY.EVENT.DEFINITION,fixing event can be set to run during COB on the fixing date. However, for a particular variant of Accumulatorswhich is based on US Stocks, the price may be available only the day after the fixing date. Hence the Fixing eventshould not run during the COB on the fixing date but rather the users would prefer to run this Online afterreceiving the correct price. In such situations, a variant can be created for Accumulators on US stocks and thisfield can be set to ONLINE.

        Parameters:
        Returns:
        TField - The CobPhase
