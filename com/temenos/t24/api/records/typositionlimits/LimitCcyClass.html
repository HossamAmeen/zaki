      • getIntradayLimAmt

        public TField getIntradayLimAmt()
        Returns the IntradayLimAmt

        Field description: An user defined field that contains the intraday limit amount for the limit currency. This amount is automatically assigned to the respective TY.INTRADAY.LIM.CCY record when TY.POSITIONS.LIMITS record is authorized. The intraday position during a deal capture would be validated against the amount defined here and in case of a breach, overrides are raised in respective applications. This field is mandatory when a currency is defined in LIMIT.CCY field.

        Validation:

        Should be a valid amount. Mandatory when LIMIT.CCY is defined.

        Parameters:
        Returns:
        TField - The IntradayLimAmt

      • getLimitCcy

        public TField getLimitCcy()
        Returns the LimitCcy

        Field description: An user defined field that contains the currency for which intraday and overnight limits are to be monitored. Multiple currencies could be defined and for each currency intraday and overnight limit amounts could be monitored. Only currencies defined at company level could be re-defined at dealer level. Example : GBP currency could be defined in US0010001.CCY.POSITION.D1 only when US0010001.CCY.POSITION contains GBP as a currency.

        Validation:

        Should be a valid currency. A currency defined at dealer level should already have been defined at company level.

        Parameters:
        Returns:
        TField - The LimitCcy

      • getOvnLimitAmt

        public TField getOvnLimitAmt()
        Returns the OvnLimitAmt

        Field description: An user defined field that contains the overnight limit amount for the limit currency. The overnight position for the limit currency would be validated against the amount defined here and in case of a breach, the exceptions are raised in a report during cob. This field is mandatory when a currency is defined in LIMIT.CCY field.

        Validation:

        Should be a valid amount. Mandatory when LIMIT.CCY is defined. Should be less than intraday amount defined in INTRADAY.LIM.AMT field.

        Parameters:
        Returns:
        TField - The OvnLimitAmt
