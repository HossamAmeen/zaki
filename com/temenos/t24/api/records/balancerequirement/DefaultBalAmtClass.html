      • getDefChargeAmt

        public TField getDefChargeAmt()
        Returns the DefChargeAmt

        Field description: Specifies the charge to be made on Account Balances less than that specified in corresponding field DEFAULT.BAL.AMT. If various levels of charge are to be applied up to various Balance limits, multivalue fields DEFAULT.BAL.AMT and DEF.CHARGE.AMT are used to specify these in ascending balance sequence. The amounts and charges specified in fields DEFAULT.BAL.AMT and DEF.CHARGE.AMT are in local currency and apply to accounts in currencies not specifically covered by CURRENCY. It is possible to specify local currency as a value for multivalue field CURRENCY. Validation Rules: 1-14 numeric characters plus a decimal point. Multivalue field associated with DEFAULT.BAL.AMT. (Mandatory input if input made to corresponding field DEFAULT.BAL.AMT, otherwise not allowed.)
