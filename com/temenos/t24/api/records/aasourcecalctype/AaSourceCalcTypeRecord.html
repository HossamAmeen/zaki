      • getMidAccountClose

        public TField getMidAccountClose()
        Returns the MidAccountClose

        Field description: This field denotes whether the interest needs to be calculated for the month, if the arrangement was closed on or before the specified Calculation End Day. If set to NO, then no interest will be applied if the account was closed on or before the Calculation End Day. Setting the field to BLANK will allow the interest according to the minimum balance criteria. Here interest will not be paid even if closure happened on the Calc End Day becaue the End of day balance will become 0.

        Validation Rules:

        Optional input. BLANK or NO. Can only be set to NO if the BALANCE.TYPE is set to MINIMUM and for Monthly calculation Default = BLANK

        Parameters:
        Returns:
        TField - The MidAccountClose

      • getMidAccountOpen

        public TField getMidAccountOpen()
        Returns the MidAccountOpen

        Field description: This field denotes whether the interest needs to be calculated for the month, if the arrangement was opened after the specified Calculation Start Day for the MINIMUM interest type. If set to NO, then no interest will be applied if the account was opened after the Calculation Start Day. Setting the field to BLANK will allow the interest according to the minimum balance criteria

        Validation Rules:

        Optional input. BLANK or NO. Can only be set to NO if the BALANCE.TYPE is set to MINIMUM and for Monthly calculation Default = BLANK
