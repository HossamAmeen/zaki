      • getCgTxnBaseId

        public TField getCgTxnBaseId()
        Returns the CgTxnBaseId

        Field description: This field will contain the id of the CG.TXN.BASE record. The id is in three parts separated by a dot character ".". The first part contains the underlying CUSTOMER id, the second part contains the group to which the portfolio belongs, as defined in PORTFOLIO.GROUPING, therefore indicating the id of the portfolio to which the security belongs. If portfolio groups are not defined then the system will automatically set them up with each portfolio belonging to a unique group with the same id as the portfolio. The third part identifies the security itself. For example: 12435.12435-1.000123-000 Where 12435 is the CUSTOMER id, 12435-1 indicates the portfolio group and 000123-000 indicates the security held by the customer.
