      • calculateCharges

        public TaxData calculateCharges(ChargeCode chargeCode, ChargeCustomer chargeCustomer, DealAmountAndCurrency dealAmountAndCurrency, CrossRate crossRate)
        Calculates the 'charge' amount in both local and deal currency for many types of transaction, like loans, deposits etc. This method will return a TaxData object which can be used to raise accounting entries.
        T24 Details: This invokes the T24 routine CALCULATE.CHARGE which uses information from the FT.CHARGE.TYPE table.
        Parameters:
        chargeCode - Required. Details about the charge and it's period start and end date for the calculation.
        chargeCustomer - Optional. Use this object to set input parameters relating to a customer.
        dealAmountAndCurrency - Required. Use this object to set input parameters relating to a dealAmount and its currency.
        crossRate - Optional. Use this object to set input parameters relating to a rate.
        Returns:
        TaxData - s the TaxData.
        Throws:
        T24CoreException if parameters are not set or invalid.
