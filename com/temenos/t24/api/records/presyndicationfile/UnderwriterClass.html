      • getSubsBrought

        public TField getSubsBrought()
        Returns the SubsBrought

        Field description: Extent of subscription (participation) brought in by the underwriter. If any value is entered in this field, equivalent participations need to be recorded. In other words, aggregate of Amount Sought by all the participants should be greater than (or equal to) the aggregate subscriptions brought by all the underwriters within each SL.CURRENCY multi-value set. Validation Rules: 1-14 numeric characters plus a decimal point. Amount entered should be greater than zero. Aggregate Amount.Sought should be greater than or equal to aggregate Subs.Brought within each SL.Currency multi-value set.

        Parameters:
        Returns:

      • getUnderwriter

        public TField getUnderwriter()
        Returns the Underwriter

        Field description: Identification number or Mnemonic assigned to the Underwriter. When underwriting details are entered in the Presyn file, this set of fields could be used to record the name of the underwriter, date of the agreement, amount and percentage underwritten and fee payable for underwriting. Validation Rules: 3-10 alphanumeric customer mnemonic or 1-10 numeric character customer number Must be a valid ID on the customer table If SL.FUNCTION is P (Only Participant), only OWN.CUST.REF in SL.PARAMETER would be accepted. Multi-value field to facilitate recording details of more than one underwriter

        Returns:

        public TField getUwCommAmt()
        Returns the UwCommAmt

        Field description: Amount of underwriting commission payable based on amount underwritten and percentage fee payable. System maintained field. Value in this field would be updated with fresh underwriting and change of fee percentage. Amount in this field is expressed in SL.CURRENCY The net commission payable would be debited to the Uw Fee Account and credited to the respective underwriter's account on creation of FACILITY Validation Rules: System maintained field Input by user is not allowed
