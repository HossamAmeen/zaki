        A,pre,postAdd the text pre before TEMP and append the text post.

        I,convPerform a DataBasic input conversion (ICONV) using conv as the conversion mask.
        L,application,field nameLookup record keyed by the data in TEMP in the T24 application "application" and return the data in field field name. I.E. L,DX.TRADE,BUY.SELL Would extract the BUY.SELL field data from a DX.TRADE record and store the result in the storage variable.
        M,maskPerform a DataBasic format mask i.e. M,R#10 would apply R#10 mask to TEMP
        O,convPerform a DataBasic output conversion (OCONV) using conv as the conversion mask.
        R,old,newReplaces the current value of TEMP with new if the value of TEMP = old.
        S,sub,parameter(s)Call and external DataBasic subroutine sub, and set TEMP to the result. Note: The subroutine header should take the following syntax SUBROUTINE sub(RESULT,INPUT.DATA,,parameter(s))
        T,char,modeTrims character char from TEMP. Apply the DataBasic TRIM function on character char using mode.

        BRemove both leading and trailing occurrences of character

        Field description: A list of extraction tasks that document what data is to be extracted from the input record or the 'special' record and held in the storage variable. These tasks fall in to two categories "named" and "un-named". Unnamed extractions take data directly from a location with no reference to the field/record names. Named extraction us the Temenos&trade; T24 STANDARD.SELECTION record to extract data from a location within the record. "Un-named" Extractions

        Extraction CodeMeaning
        A,x,y,z Extract the data from attribute<x,y,z> of the standard input variable.
        AS,x,y,zAs A, but take the data from the special input variable.
        C,textStore the constant text.
        S,x,yTake a string of y characters, starting at x, from standard input variable.
        SS,x,y As S, but take the data from the special input variable.
        "Named" Extractions
        Extraction CodeMeaning
        N,application,field name,y,z Extract the data from attribute<field name,y,z> of the standard input variable. Where the format of the incoming record is known. i.e. N,CUSTOMER,SHORT.NAME becomes N,,2 which will extract the SHORT.NAME data from the standard input variable.
        NS,application,field name,y,zAs A, but take the data from the special input variable.
        NV,application,field name,value nameExtracts data from a Value pair array. Specify the name in value name and data will be extracted from the next field. Takes the data from the standard input variable.

        Field description: A list of placement tasks that document where the data held in the standard storage variable. Data can be stored in the standard output record and/or the special input record. These tasks fall in to two categories "named" and "un-named". Unnamed placements take the data and store it directly into a location with no reference to the field/record names. Named placement uses the Temenos&trade; T24 STANDARD.SELECTION record to store the data into a location within the specified record. The data to be placed is held in the temporary storage variable, known as TEMP.

        "Un-named" Placements
        Placement CodeMeaning/Effect
        A,x,y,zAppend the data from TEMP into attribute<x,y,z> of the standard output variable.
        AS,x,y,zAs A, but append the data to the special variable.
        R,x,y,z Replace attribute<x,y,z> of standard output with TEMP
        RS,x,y,zReplace attribute<x,y,z> of special with TEMP.
        S,x,yPlace the data from TEMP into string standard outputstarting at character x and placing a maximum of y characters.
        SS,x,ySame as S, but place the data in the special variable.
        P,x,y,zAdd the data from TEMP into attribute<x,y,z> of the standard output variable
        PS, tox,y,zSame as P, but add the data in TEMP to the special variable
        M,x,y,z Subract the data in TEMP from attribute<x,y,z> of the standard output variable
        MS,x,y,z Same as M , but subtract the data in TEMP from special variable
        X,x,y,zMultiply the data in TEMP with attribute<x,y,z> of the standard output variable
        XS,x,y,z Same as X , but mulitply the data in TEMP to the special variable
        D,x,y,z Divide attribute<x,y,z> of the standard output variable by TEMP
        DS,x,y,zSame as D , but divide the special by the data in TEMP
        "Named" Placements
        Placement CodeMeaning/Effect
        NA,application,field name,y,zAppend the data from TEMP into attribute<field name,y,z> of the standard output variable. i.e. NA,DX.TRANSACTION,CONTRACT.CODE becomes N,,6 which will place TEMP data in the CONTRACT.CODE field output variable.
