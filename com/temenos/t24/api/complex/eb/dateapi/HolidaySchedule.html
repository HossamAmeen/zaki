com.temenos.t24.api.complex.eb.dateapi

Class HolidaySchedule

Packaged in : EB.DateApi.jar

Represents objects from the HOLIDAY table in T24. Each HolidaySchedule contains information about a years worth of holiday and work days in a particular REGION.
