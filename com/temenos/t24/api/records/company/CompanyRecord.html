        public void addAccBknoPrefix(TField param)
        Appends the specified AccBknoPrefix to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Identifies, in countries like Belgium and Luxembourg, the Bank Prefix which is part of the local account number. In countries like Belgium and Luxembourg, each bank receives a range or a unique identification numbers. This may be used with the following checkdigit types: '1' - 3 numeric '2' - 2 numeric '3' - any value '4' - 2 numeric @routine - up to 6 numeric Note : Once a customer account record has been authorised in this company, this field may no longer be changed. Validation Rules: Up to 6 numeric characters. Multivalue field.
