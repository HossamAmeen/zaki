Class Contract

Packaged in : AA.ContractApi.jar

The Contract class allows access to balance movements, payments, property conditions, payment schedules and all infomation related to Contract details record.

        public List<PaymentSchedule> buildPaymentSchedule(TDate startDate, TDate endDate, TNumber noOfCycles, List<String> paymentTypes)
        Returns the Contract's payment schedules for the given period, number of cycles and payment types.

        T24 Details: This invokes the T24 routine AA.GET.ARRANGEMENT.CONDITIONS
        Parameters:
        startDate - Optional - Start date from which payment date cycling is required, defaults to the Contract's start date.
        endDate - Optional - End date until which the payment schedule will be built, defaults to the Contract's end date.
        noOfCycles - Optional - number of cycles required in payment schedule. Eg 5 , defaults to all the cycles in the schedule.
        paymentTypes - Optional - Payment types. a valid payment type. eg. ACTUAL','LINEAR' , criteria ignored if not provided.
        Returns:
        List<PaymentSchedule> - the Contract's payment schedules for the given criteria.
        Throws:
        T24CoreException if contractId is invalid

        Returns the Contract's bill Ids for the given bill date, payment date, bill type, payment method, bill status, settlement status, aging status, next ageing date and repayament reference.

        T24 Details: This invokes the T24 routine AA.GET.BILL
        Parameters:
        billDate - Required - Effective date when the bill was issued, eg '20160322'
        paymentDate - Required - Scheduled date for repayment, eg '20160325'

        billStatus - Required - BillStatus , must be one of ISSUED, DUE, AGING or SETTLED
        settlementStatus - Required - SettlementStatus of the Bill, must be one of UNPAID or REPAID
        agingStatus - Required - AgingStatus of the Bill, must be a valid entry in EB.LOOKUP for the Virtual table AA.OVERDUE.STATUS, eg 'DEL'
        nextAgeDate - Required - Next Aging Date of the bill , eg '20160322'
        repaymentReference - Required - RepaymentReference of the bill, eg 'AAACT093573MG59Y2M-20091201'
        Returns:
        List<String> - the Contract's bill Ids for the given criteria.
        Throws:
        T24CoreException if contractId is invalid.
