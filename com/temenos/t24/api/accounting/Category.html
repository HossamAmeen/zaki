com.temenos.t24.api.accounting

Class Category

Packaged in : AC.CategoryApi.jar

Provides methods to get profit and loss entries for the given category. The member variable for this class is categoryId and this must be set for most of the methods to work correctly.

      • getProfitAndLossEntryIds

        public List<String> getProfitAndLossEntryIds(TDate startDate, TDate endDate)
        Returns the Profit & Loss entry IDS (called CATEG entries in T24 and stored in CATEG.ENTRY table) related to the categoryId, between the given dates.
        T24 Details: This invokes the T24 routine GET.CATEG.MONTH.ENTRIES.
        Parameters:
        startDate - Optional. If startDate is not provided it will default to starting date of the financial year (FINANCIAL.YEAR.END in COMPANY table).
        endDate - Optional. If endDate is not provided it will default to the current system date.
        Returns:
        List<String> - the list of ids from the CATEG.ENTRY table that match the given criteria.
        Throws:
        T24CoreException if categoryId is not provided or if endDate <= startDate.
