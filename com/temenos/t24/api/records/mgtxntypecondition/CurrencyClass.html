      • getMaxIntTol

        public TField getMaxIntTol()
        Returns the MaxIntTol

        Field description: The upward tolerance for the initial interest that is to be defaulted to the MG deal. The initial 'applicable rate' at the time of striking the deal is stored in the MG record. For fixed rate contract it is the rate input and for floating rate contracts it is the rate in the respective key plus spread at the time of striking the deal. Any upward rate increase during the life of the MG will be restricted to this percentage on the initial interest so derived. For more details refer help on MAX.INT.TOL in the MG application. Validation Rules: Standard T24 percentage format. Negative values not allowed. Value input cannot exceed 100. Optional Input. Either MAX.INT.TOL or MIN.INT.TOL alone can be defined if so required.
