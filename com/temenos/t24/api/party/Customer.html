      • getLimit

        public LimitAmount getLimit(String limitReferenceId, String currencyId, TBoolean isUnavailableLimit, TBoolean isOverUtilisedAmount)
        Returns the Customer's allocated and available limit amounts for the given limit reference and currency.
        The caller has the option of activating the available marker and disabling the overutilised amount for the limit.
        If no limit is available for the customer, it will check for the liability limit (parent customer's limit).

        T24 Details: This invokes the T24 routine T24CustomerServiceImpl.GetAccountLimitAmts.
        Parameters:
        limitReferenceId - Required. Id of the limit reference.
        currencyId - Required. Currency in which the amount is to be returned.
        isUnavailableLimit - Optional. Should the available marker flag in the limit record be considered? Defaults to false.
        isOverUtilisedAmount - Optional. Should the overutilised limit amount should be included? Defaults to true.
        Returns:
        LimitAmount - the Customer's allocated and available limit amounts.
        Throws:
        T24CoreException on null or invalid customer Id.

      • getSwiftAddress

        public SwiftAddress getSwiftAddress(String companyCode, TNumber sequenceNo, String preferredLanguage)
        Returns the Customer's SWIFT communication address for the given company code, sequence number and preferred language.
        The data is obtained from DE.ADDRESS and the critera are applied progressively to obtain the most appropriate match.
        An exception is raised if no matching address can be found.

        T24 Details: This invokes the T24 routine T24CustomerServiceImpl.getSWIFTAddress.
        Parameters:
        companyCode - Optional. Company code, defaults to the current company.
        sequenceNo - Optional. Sequence No, defaults to '1' meaning that the 1st address is required.
        preferredLanguage - Optiona. Id of preferred language for the address, defaults to 1 (English).
        Returns:
        SwiftAddress - the Customer's SWIFT communication address and raises an exception if not found.
        Throws:
        T24CoreException on null or invalid customer Id.
