      • getAmortPeriod

        public TField getAmortPeriod()
        Returns the AmortPeriod

        Field description: Field to accept the period up to which the charge is to be amortized. This field can be input only if amortization is set to YES. For eg. if the charge is to be amortized over a period of 6 months, a value 6M is to be input in this field ; if the charge is to be amortized over a period of 2 years, a value 2Y is to be input in this field and in case the user desires to amortize the charge over the life of the contract a value "C" is to be input in this field. Validation Rules: Can be input only if amortization is set to YES. Accepted values are 1M,2M....11M, 1Y, 2Y.... and "C" in case the amortization is to be done for the entire life of the contract.
