      • getCustComponent

        public TField getCustComponent()
        Returns the CustComponent

        Field description: Specifies the way to extract Customer reference from the field, CUSTOMER.FIELD. This field is used to specify the exact position or the means to extract the customer reference from the field, CUSTOMER.FIELD. It can be defined in the format x,y. Example 1: If CUSTOMER.FIELD has the value : 10102520070101 (Customer number appended with Date), the field CUST.COMPONENT has to be defined with the value 1,6 to denote that customer number exists from first position until sixth position. Example 2: If CUSTOMER.FIELD has the value : 20070101.101025.01, CUST.COMPONENT should be defined as .,2 where “.” as the delimiter, the second value has to be extracted as the customer number. This is an Optional Field.
