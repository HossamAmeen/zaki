NONRED In NONRED type of loans, there is no regular repayment of principal, but only interest. The redemption of Principal will be done on maturity date of the loan. In All in One account loans, the validations relating to Interest Only and Maximum Period interest only will not happen for this type of loan. 4. OTHER Non structured repayments are handled under this type.
