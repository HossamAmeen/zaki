      • getGroupLineFrom

        public TField getGroupLineFrom()
        Returns the GroupLineFrom

        Field description: The line number of the lower end of a reporting group. This field forms part of a multi-value set with GROUP.LINE.TO and GROUP.DESC. These fields are used to divide the report lines into groups for reporting purposes. For example of lines 1 to 0999 are for Assets and 1000 to 1999 are for liabilities then this definition can be specified here in two multi-value sets. MI will automatically add the description of the group (in field GROUP.DESC) to each MI.LINE record. This can subsequently be referenced in enquiries and reports and used to break up the final MI output into reporting ranges. Ranges must be input in ascencing sequence. Validation Rules: Must be numeric and less than 9999 Must be greater than any earlier group. (i.e. line groups must be input in ascending sequence.

        Returns:
