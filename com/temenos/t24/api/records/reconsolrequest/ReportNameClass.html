      • getRatesMkr

        public TField getRatesMkr()
        Returns the RatesMkr

        Field description: Indicates where the exchange rates are to be taken from. Specifies the Currency file from which the exchange rates are to be taken. DEFAULT means use the Currency file specified on the Company Consol record. COMPANY means take the rate between the local currency for the company and the reporting currency for the Currency file for that Company. Note where more than one Currency file for the same local currency the file to be used is indicated on the Company Consol record. SPECIAL means that the rates on the RE.PRT.CCY.RATE file for the record indicated in the field Spec.Rate.Id (field 8.1) are to be used. Note wher a rate is found to be missing at report time then a cross rate using the Master Company Currency file is used. The Master Company is the first company set up. Validation Rules: Must be DEFAULT, COMPANY or SPECIAL. Multivalue field associated with fields 3.1 to 6.1 and 8.1. Mandatory input. : Must be DEFAULT, COMPANY or SPECIAL.

        Parameters:
        Returns:
        TField - The RatesMkr
