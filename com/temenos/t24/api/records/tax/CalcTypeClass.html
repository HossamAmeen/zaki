      • getCalcType

        public TField getCalcType()
        Returns the CalcType

        Field description: The method of calculation to be used for the associated currency. This field specifies whether separate calculations are to be made for each band of the base amount, whether only one is calculated which is dependent on the level of the base amount. Example: Assume a deal of Principal 100,000 USD where tax is calculated based on the Principal. The tax record specifies an applicable rate of 2 per cent up to 10,000 and 1.5 percent over 10,000. If Band is specified for all sub value groups, 2 calculations will be performed, i.e. 2% on 10,000 1.5% on 90,000 If Level is specified for all sub value groups, 1 calculation will be performed, ie. 1.5% on 100,000 A change can be made in one band to the next from BAND to FLAT but not vice versa Validation Rules: LEVEL or BAND or any abbreviation thereof. Mandatory if banded taxes are specified. No input if a flat RATE has been specified.

        Parameters:
        Returns:
        TField - The CalcType
