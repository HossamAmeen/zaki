      • getCrfType

        public TField getCrfType()
        Returns the CrfType

        Field description: Normally accruals are booked to CAL (CRF or CONSOLIDATE.ASST.LIAB) and to P&L using the P&L category code, i.e the CAL asset type is the same as the P&L category code.

        For Arrangement Architecture accruals the CAL entry can be booked using an asset type that differs from the P&L category code. This field will hold the asset type for AA accruals and if it is not populated the CRF accruals will be booked using the P&L code.
