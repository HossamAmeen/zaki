        Field description: Determines the action to be performed by the system when Escrow balance is short of cushion amount requried for disbursement to escrow payee(s). This field is associated with CURRENCY, SURP.THRESHOLD.AMT, SURPLUS.OPTION and TEST.ANALYSIS.PERIOD

      • getSurplusOption

        public TField getSurplusOption()
        Returns the SurplusOption

        Field description: Determines the action to be performed by the system when Escrow balance breaches the threshold amount mentioned in SURP.THRESHOLD.AMT and CURRENCY. Input is this field is required only if Escrow processing feature is used. This field is associated with CURRENCY, SURP.THRESHOLD.AMT, SHORTAGE.OPTION and TEST.ANALYSIS.PERIOD

        Validation rules

        Reduce Escrow payment - The excess escrow balance beyond SURP.THRESHOLD.AMT held in ESCROWBAL balance type of the loan contract will be adjusted along with the future escrow instalments Refund to customer - The excess escrow balance beyond SURP.THRESHOLD.AMT held in ESCROWBAL balance type of the loan contract will be refunded/credited to the customer account defined in SETTLEMENT property class of the arrangement

        Returns:
