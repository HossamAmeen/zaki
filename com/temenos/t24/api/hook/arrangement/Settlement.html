      • validateUserRoutine

        public void validateUserRoutine(String CallType, TStructure CallApplicationRecord, String SettlementMethod, String ArrangementAccount, String SettlementAccount, String SettlementAmount, String SettlementType, TString Exceptions)
        This interface enables the implementer to return exceptions if any after execution of the User defined logic routine.
        This interface is invoked when User rule routine is attached in AA.SETTLEMENT.TYPE record.
        T24 Details: Interface method for the USER.RULE.ROUTINE field in the AA.SETTLEMENT.TYPE table.
        It is invoked from AA.EVALUATE.SETTLEMENT.RULES to execute the routine and retrun exceptions if any.
        If any exception is returned further process of settlement is skipped.
        Parameters:
        CallType - CallType - contains Call type (RC/Settlement)
        CallApplicationRecord - CallApplicationRecord - contains Settlement Record/ RC Details
        SettlementMethod - SettlementMethod - Either DEBIT/CREDIT
        ArrangementAccount - ArrangementAccount - Arrangement Account No
        SettlementAccount - SettlementAccount - Settlement Account No
        SettlementAmount - SettlementAmount - Amount to be settled
        SettlementType - SettlementType - Record Id of AA.SETTLEMENT.TYPE
        Exceptions - Return Exceptions if any
        Returns:
        nothing
