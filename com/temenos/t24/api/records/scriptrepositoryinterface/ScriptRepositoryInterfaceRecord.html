        Field description: Indicates what needs to be done with the selection. Use UPLOAD.TO.MASTER and DOWNLOAD.FROM.MASTER for Update Activity. Use RELEASE.SPLITTER for downloading scenarios with results Test Base wise, generally intended for Regression Execution. Valid Inputs are: UPLOAD.SSA, UPLOAD.SCRIPTS, UPLOAD.RF, UPLOAD.FV, UPLOAD.SSFO, UPLOAD.ALL, DOWNLOAD.SSA, DOWNLOAD.SCRIPTS, DOWNLOAD.RF, DOWNLOAD.FV, DOWNLOAD.SSFO, DOWNLOAD.ALL, UPLOAD.TO.MASTER, DOWNLOAD.FROM.MASTER, RELEASE.SPLITTER a.
