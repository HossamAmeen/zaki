      • getMatchCondition

        public TField getMatchCondition()
        Returns the MatchCondition

        Field description: Field to indicate the matching condition to be used. If value provided, condition in matching condition will override any setup in the Check value Date, Match fields and Tolerance part of the Exp Funds Type multivalue set in ER.PARAMETER. It is a multi-value field with the associated field as 'MATCH.FUNDS.TYPE'. Validation Rules: User can define maximum of 2 multi-value sets. The value will be selected from a list with all the records defined in ER.MATCHING.CONDITION table. 1-35. Mandatory input.

        Parameters:
        Returns:
