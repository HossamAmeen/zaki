Class Currency

Packaged in : ST.CurrencyApi.jar

The Currency class provides methods for formatting amounts and calculating currency rates and amounts. The member variable for this class is currencyId.

      • calculateBuyAmount

        public ExchangeAmount calculateBuyAmount(String buyCurrency, String sellCurrency, TNumber sellAmount, String currencyMarket, TNumber exchangeRate, TDate historyDate)
        Calculates the buy amount for a currency pair, sellAmount, exchangeRate, historyDate and currencyMarket.
        If a historyDate is not provided the current date will be used.

        T24 Details: This invokes the T24 routine EXCHRATE.
        Parameters:
        buyCurrency - Required. Key to the CURRENCY table e.g. 'GBP'.
        sellCurrency - Required. Key to the CURRENCY table e.g. 'USD'.
        sellAmount - Optional. The sell amount, defaults to '1'.
        currencyMarket - Required. Key to the CURRENCY.MARKET table. E.g. 1, 2, etc.
        exchangeRate - Optional. The buyCurrency exchange rate, if not provided it will be calculated.
        historyDate - Optional. The date for which we need the exchange rate, defaults to today.
        Returns:
        ExchangeAmount - the ExchangeAmount (buy amount in both local and buy currency).
        Throws:
        T24CoreException on invalid parameters or on internal t24 errors.

      • calculateBuyExchangeRateAmount

        public ExchangeRateAmount calculateBuyExchangeRateAmount(String buyCurrencyMarket, String sellCurrencyMarket, String buyCurrency, String sellCurrency, TNumber sellAmount, TNumber spreadPercentage)
        Calculates the buy ExchangeRateAmount for the given currencyMarkets, currency pair, and sellAmount.
        The ExchangeRateAmount contains the base currency, the treasury & customer rates and amount in both buy and local currencies.
        Base Currency - The currency in which the rates are expressed Treasury rate - The rate specified by the Treasury unit to the other units within the bank.
        Customer rate - The final rate for the customer arrived at by adding the customer spread to the Treasury rate.

        T24 Details: This invokes the T24 routine CUSTRATE.
        Parameters:
        buyCurrencyMarket - Required. The market on which the buy currency is traded, key to the CURRENCY.MARKET table. E.g. 1, 2, etc.
        sellCurrencyMarket - Optional. The market on which the sell currency is traded, key to the CURRENCY.MARKET table. E.g. 1, 2, etc. Defaults to buyCurrencyMarket.
        buyCurrency - Required. Key to the CURRENCY table e.g. 'GBP'.
        sellCurrency - Required. Key to the CURRENCY table e.g. 'USD'.
        sellAmount - Required. The the amount in sellCurrency.
        spreadPercentage - Optional. The spreadPercentage, if not provided it will be calculated.
        Returns:
        ExchangeRateAmount - the base currency, the treasury & customer rates and the amounts in both buy and local currency.
        Throws:
        T24CoreException on invalid parameters or on internal t24 errors.

      • calculateForwardBidValueRate

        public ForwardRate calculateForwardBidValueRate(TNumber currencyMarket, String buyCurrency, String sellCurrency, TDate exchangeDate, TBoolean useRevalRate, TBoolean useCompanyInterpolation)
        Calculates the forward exchange bid rate for the given buy and sell currencies on the exchange date.
        If a rate for the exchange date is not available then the system will interpolate a value.

        T24 Details: This invokes the T24 routine FWDRATES.
        Parameters:
        currencyMarket - Required. Key to the CURRENCY.MARKET table. E.g. 1, 2, etc.
        buyCurrency - Required. The buy currency id, key to the CURRENCY table e.g. 'GBP'.
        sellCurrency - Required. The sell currency id, key to the CURRENCY table e.g. 'USD'.
        exchangeDate - Required. The date for which the rate is required.
        useRevalRate - Required. If true the CURRENCY field MID.REVAL.RATE will not be used when a REVAL.RATE is available.
        useCompanyInterpolation - Required. If true then the INTERPOLATION.MKR field of the REVALUATION.PARAMETER table for the company will be used to specify the interpolation method othewise interpolate as normal.
        Returns:
        ForwardRate - the forward rate.
        Throws:
        T24CoreException on invalid parameters

        public TNumber calculateRate(String buyCurrency, String sellCurrency, String currencyMarket, String outputCurrency, TDate historyDate)
        Calculates the exchange rate for a currency pair on the given date and currencyMarket.
        The outputCurrency determines which currency the rate is expressed in.
        If the historyDate is not provided the live currency record will be used for the calculation.

        T24 Details: This invokes the T24 routine EXCHRATE.
        Parameters:
        buyCurrency - Required. Key to the CURRENCY table e.g. 'GBP'.
        sellCurrency - Required. Key to the CURRENCY table e.g. 'USD'.
        currencyMarket - Required. Key to the CURRENCY.MARKET table. E.g. 1, 2, etc.
        outputCurrency - Required. The currency id in which the rate will be output, key to the CURRENCY table e.g. 'GBP'. Must be either the buy or the sell currency.
        historyDate - Optional. The date for which we need the exchange rate, defaults to today.
        Returns:
        TNumber - the exchange rate.
        Throws:
        T24CoreException on invalid parameters or on internal t24 errors.

      • getCurrencyRates

        public List<CurrencyRate> getCurrencyRates(TDate historyDate)
        Returns a list of the Currency's rate information for the year of the given historyDate.
        There will be a separate element in the list for each currency market that the currency is traded in.
        If a historyDate is provided the system will return the latest rates for that year.
        If the historyDate is not provided then the system will return the current rates.
        If a rate is not available for the year it will return the rates for the closest prior year.

        T24 Details: This invokes the T24 routine GET.CCY.HISTORY.
        Parameters:
        historyDate - Optional. The date of the required year, defaults to the current date.
        Returns:
        List<CurrencyRate> - the CurrencyRates.
        Throws:
        T24CoreException on invalid or empty parameter

      • getRoundAmount

        public TNumber getRoundAmount(TNumber amount, String roundingRuleId)
        Returns the Currency's formatted amount based on the given roundingRuleId.
        If the roundingRuleId is not provided then the amount will be formatted according to the currencyId member variable.
        If neither roundingRuleId nor currencyId member variable are set the amount will be formatted to two decimal places.

        T24 Details: This invokes the T24 routine EB.ROUND.AMOUNT.
        Parameters:
        amount - Required. The amount to be rounded.
        roundingRuleId - Optional. Key to the EB.ROUNDING.RULE table. E.g NATURAL, TAX.UP, etc. Ignored if not provided.
        Returns:
        TNumber - a formatted currency amount.
        Throws:
        T24CoreException if no amount passed.
