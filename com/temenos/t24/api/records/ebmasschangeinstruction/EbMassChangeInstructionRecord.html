      • getEffectiveDate

        public TField getEffectiveDate()
        Returns the EffectiveDate

        Field description: The field defines the "Effective Date" which the Mass Change Instruction should be executed. It can be Forward Dated or Back Dated as long as it is enabled and supported in the underlying Business Operation.
