        public void addSystemId(TField param)
        Appends the specified SystemId to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Identifies the System.Id for which Netting details to be created. Details entered here along with COUNTERPARTY, SYSTEM.ID, REC.CORR.BANK, SEND.CORR.BANK, VALUE.DATE, CURRENCY and OPERATION.CODE is be used to select criteria for propagation to the NETTING file to generate MT102 / MT203 instead of MT103 / MT202 respectively. To create netting for FOREX application CURRENCY, COUNTER.PARTY, and VALUE.DATE is alone used to for selection Criteria and Input in this field not allowed. This field is made as multi value field for future use. Validation Rules: FT or FX Mandatory input.
