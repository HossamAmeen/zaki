      • getHybridAcctType

        public TField getHybridAcctType()
        Returns the HybridAcctType

        Field description: This field will denote that current acct method being defined follows the Hybrid method of IFRS accounting when attached to any contracts. In Hybrid method of IFRS accounting, on the firsday of a loan disbursement the contract will be measured based on Fairvalue using the Market Key. On subsequent days the loan is measured using amortised cost method

        Options field with allowed values as "YES" or "NO". When a value of "YES" is provided then Multivalue set of ACCT.HEAD.TYPE should have "FAIRVALUE" defined first followed by "AMORTISED". Othercase validation error would be raised. When a value of "NO" or null is specified then any existing definition allowed but "FAIRVALUE" followed by "AMORTISED" will be stopped.

        Parameters:
        Returns:
        TField - The HybridAcctType
