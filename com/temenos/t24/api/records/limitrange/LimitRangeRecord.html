        public void addDiscrValue(java.lang.CharSequence param)
        Appends the specified DiscrValue to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Discrete values of COMPANY or CURRENCY codes. This field defines discrete values of either CURRENCY or COMPANY (whichever is defined in COMP.OR.CCY). Validation Rules: 2-9 Alphanumeric characters Multi value field Optional input No duplicate codes allowed If COMP.OR.CCY is specified as COMPANY, the value can be input as the 2-3 character company mnemonic, or the full 9 character company code. The company code must exist on the COMPANY file. If COMP.OR.CCY is specified as COMPANY, the company should be a financial company. Branches are not allowed. If COMP.OR.CCY is specified as CURRENCY, the value can be input in standard currency format, and must exist on the CURRENCY file. Both DISCR.VALUE and a range of values (using START.VALUE and END.VALUE) cannot be specified together.
