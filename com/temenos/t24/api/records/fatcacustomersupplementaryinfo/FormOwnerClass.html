      • getFormOwner

        public TField getFormOwner()
        Returns the FormOwner

        Field description:

        This field is used to maintain document information for each owner.

        If the owner is not given, it will be assumed that the documents pertain to the Primary Customer. If these documents are being recorded for Beneficial and other owners, then the owner ID has to be recorded here.

        It must be noted that for classification of accounts under FATCA, the documents have to be obtained from beneficial, joint and other owners as well.

        The fields from FORM.TYPE to EXP.DATE will be associated sub value fields within this FORM.OWNER multi-value field.

        Validation rules: FATCA CUSTOMER SUPPLEMENTARY INFO ID or HOLDER REF value can be specified here.

        Parameters:
        Returns:
        TField - The FormOwner
