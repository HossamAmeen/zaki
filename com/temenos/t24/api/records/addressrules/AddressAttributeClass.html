      • getMinLength

        public TField getMinLength()
        Returns the MinLength

        Field description: Indicates the minimum number of characters. If mandatory then set minimum length. By defining a minimum (i.e. 1 character), it implies that it is considered a mandatory address attribute. Without defining a minimum length for an attribute, the system will consider the respective address attribute as optional for input. Validation Rules: It should validate the name defined in ATTRIBUTE field with the SS record field length and if less than the min length of the field should raise error. If its set as mandatory field in SS then populate MIN.LENGTH as 1.

        Returns:
