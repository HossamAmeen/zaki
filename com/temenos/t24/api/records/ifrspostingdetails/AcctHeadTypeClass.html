      • getAcctHeadType

        public TField getAcctHeadType()
        Returns the AcctHeadType

        Field description: Holds the asset types defined in the IFRS.ACCT.METHODS i.e AMORTISED, FAIRVALUE, DISCLOSURE, IMPAIR.AMORTISED, UNWIND or IMPAIR.AMC.ADJUST which is applicable for both IAS39 and IFRS9 regulations. Also holds the hedge asset types HEDGED.EFFECTIVE, HEDGING.EFFECTIVE, HEDGED.INEFFECTIVE, HEDGING.INEFFECTIVE, HEDGED.CLOSED.RELATION, HEDGING.CLOSED.RELATION In addition, this field holds the asset types AMORTISED-STAGE1, AMORTISED-STAGE2, FAIRVALUE-VALUE1, FAIRVALUE-VALUE2 specific to IFRS9 regulation.

        Validation Rules: : Hedge related asset types are allowed only when 'IH' product is installed in the company. If HEDGED.EFFECTIVE is defined, then system validates for HEDGING.EFFECTIVE also to be defined Asset types related to IFRS 9 are allowed only when 'I9' product is installed in the company, ST.REGULATORY.PARAMETER configured for IFRS record and IFRS.PARAMETER is being set for the current company.

        Returns:
