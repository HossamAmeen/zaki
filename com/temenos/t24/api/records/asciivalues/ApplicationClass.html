        public TField getAppValueTbl()
        Returns the AppValueTbl

        Field description: This field is used to specify which ASCII.VAL.TABLE record is to be used for validation for the application specified in field APPLICATION. It is used in conjunction with field APPLICATION when defining special validation for particular applications where the standard table in field DEF.VALUE.TBL is not sufficient. Validation Rules: 1-20 alphanumeric (type 'A') characters which must be the key to a record on the ASCII.VAL.TABLE table. Multivalue field associated with fields APPLICATION, APP.LANG and APP.USE.VAL. Mandatory input if corresponding field APPLICATION entered otherwise optional input.

      • getApplication

        public TField getApplication()
        Returns the Application

        Field description: This field defines the application for which a non-standard ASCII.VAL.TABLE record is to be used for validation. This is for special cases where the standard table defined in field DEF.VALUE.TBL is not sufficient. Validation Rules: Up to 35 characters which must be the key to a record on the PGM.FILE file with one of the following types: 'H', 'U', 'L', 'T', 'W' or 'A'. Multivalue field associated with fields APP.VALUE.TBL, APP.LANG and APP.USE.VAL. Mandatory input if corresponding field APP.VALUE.TBL entered otherwise optional input.
