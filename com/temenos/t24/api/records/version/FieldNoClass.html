        public TField getExpansion()
        Returns the Expansion

        Field description: Indicates whether the Field Number previously defined can be expanded when using this Version in an on-line session. This field should only be used when FIELDS PER LINE (Field 3) is equal to MULTI and the User does not want to allow expansion for multivalue and sub-value fields. If this field is left blank, the multivalue or sub-value field can then be expanded on-line when using this Version [by means of the '>' or ')' sign] and all existing values will be displayed. This field can only be used for multivalue and sub-value fields. Only the value(s) defined in the various occurrences of FIELD NO. will be displayed, even if the application record contains more values. Example: Let us suppose that Field 12 (OTHER OFFICER) from the CUSTOMER file has been defined as follows : 12.1 with column 1, 12.2 with column 40 and the field Expansion has been left blank. In this case, column 1 will be used for Fields 12.1, 12.3, 12.5 etc... while column 40 will be used for Fields 12.2, 12.4, 12.6 etc... Validation Rules: NO The User will not be able to expand the multivalue or sub-value field. blankIt can be expanded by the User during the on-line session when using this Version. Multivalue field associated with Fields 13 to 23. (Optional input. Default value = blank.) Input in this field is only allowed when FIELDS PER LINE (Field 3) contains Multi. Input of NO in this field is mandatory when the field defined in FIELD NO. (Field 13) is a Local Reference data element and FIELDS PER LINE contains Multi. The system will not accept a mix of Expansion and No Expansion of fields with the same field number or fields in the same group of associated fields. For example, Field 12 (OTHER OFFICER) from the CUSTOMER file cannot be defined as 12.1 with expansion and 12.2 without expansion. In the same way, the associated RELATION fields of the CUSTOMER file cannot be defined as 7.1 with expansion, 8.1 also with expansion and 9.1 without expansion.
