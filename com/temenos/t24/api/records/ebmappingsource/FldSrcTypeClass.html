      • getFldDataSrc

        public TField getFldDataSrc()
        Returns the FldDataSrc

        Field description: EB.MAPPING.SOURCE FLD.DATA.SRC This field holds data according to the value specified in the FLD.SRC.TYPE. The field FLD.SRC.TYPE provides the source data type for FIELD mapping and this field has the exact source based on the source type. Validations This field can take the following values: 1. If FLD.SRC.TYPE is ‘PW.ACTIVITY’ then this field can hold a PW.ACTIVITY ID or literal ‘CUSTOMER’. If the value ‘CUSTOMER’ is specified then the FIELD mapping source data is obtained from the Customer.no mentioned in the field CUSTOMER in PW.PROCESS. Also this field can hold a PW.ACTIVITY ID along with UNIQUE.NAME (for ex., CREATE.CUSTOMER.MYUNIQ where CREATE.CUSTOMER denotes the PW.ACTIVITY id and MYUNIQ denotes the UNIQUE.NAME attached to the PW.ACTIVITY). This way of definition is to enable the process at runtime to refer to the latest instance of the PW.ACTIVITY. This is required since there can be multiple instances of the PW.ACTIVITY due to the implementation of iteration patterns like Structured & arbitrary loops. 2. If FLD.SRC.TYPE is ‘ROUTINE’ then @ROUTINE will be attached here. (Need to define an EB.API record for the @ROUTINE) 4.If FLD.SRC.TYPE is ‘NONE’ then this field is left NULL

        Parameters:
        Returns:
        TField - The FldDataSrc

      • getFldSrcType

        public TField getFldSrcType()
        Returns the FldSrcType

        Field description: EB.MAPPING.SOURCE FLD.SRC.TYPE This field is similar to ID.SRC.TYPE except that this field specifies from where we should get the source data for FIELD mapping. Validations 1. Mandatory field 2. Valid values are: 1. PW.ACTIVITY Uses the common variable PW$ACTIVITY.TXN and reads the source record based on the values in TARGET & TRANS.REF in PW.ACTIVITY.TXN. The name of the PW.ACTIVITY is specified in the field FLD.DATA.SRC. 2. ROUTINE This means that a user routine is attached to the field FLD.DATA.SRC and the routine will return the source data for mapping. 3. NONE This denotes that there is no source data for mapping and we are going to map only constants and literals into the Target application.

        Parameters:
        Returns:
        TField - The FldSrcType
