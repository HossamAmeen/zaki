      • addTextData

        public void addTextData(TField param)
        Appends the specified TextData to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Specifies an element of text, or field whose content is to form part of the narrative being defined. Contains either Field names,text strings or a new line indicator. Field Names - A field name must exist on the Standard Selection record of FMT.FILE either as User or System defined. Any type "D", "I" or "J" field may be used. Text - A text string may be used and can include any characters except double quotes ie( " ). The text string however must be surrounded by double quotes, ie( "text string" ) New Line - A new line may be thrown in the narrative by specifying the sequence [NL] The text may be defined in various languages, the appropriate language will be used where possible according to the output language, if not found the English will be used. When the Narrative is constructed spaces will be trimmed. This means that leading, trailing spaces are removed, elements in the sentence are separated by single spaces. Care should be taken when using narratives in output sent via SWIFT, to only use acceptable characters according to the SWIFT standards. Validation Rules: Alphanumeric field Sub valued where the sub valued set uses the multi Language application Input must be either - a valid field of TYPE D,I,J in the STANDARD.SELECTION record of the FILE - Any string of TEXT enclosed in "" - The characters [NL] to signify a new line is required

        Parameters:
        param - the TextData to append
        Returns:
        nothing
