        public TField getPrdAdjMarginRate()
        Returns the PrdAdjMarginRate

        Field description: Multi-valued optional field to allow modification of the Lower Advance Ratio based on the currency of the asset for the associated product. It will allow plus (+) or minus (-) followed by a number between 0 and 100. This will amend the Lower Advance Ratio for assets in this product valued in the currency by the number of percentage points either increasing ‘+’ or decreasing ‘-‘ depending on the sign of the number. Input into this field should be ‘+’ or ‘-‘ followed by a number between 0 and 100. Validation Rules: Value needs to be between 0 to 100
