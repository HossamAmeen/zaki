      • getBaseCcy

        public TField getBaseCcy()
        Returns the BaseCcy

        Field description: This is the currency for which the rate is quoted. This field contains the currency for which the rates in the fields Exch.rate and Rpt.rate (fields 5 and 6) are quoted. This field is used in conjunction with the field Quote.Code (field 4), which specifies the number of units that are quoted. The rate enter in the field Exch.Rate (field 5) is the amount in the non base currency for X of the base currency. Where X depends upon the value in the field Quote.Code (field 4) as follows - Value Meaning 0 or Blank 1 UNIT of the base currency 1 10 UNITS " " 2 100 UNITS " " 3 1000 UNITS " " 4 10000 UNITS " " 5 100000 UNITS " " Thus the following examples with a reporting currency of BEF. - Local.ccy Base.ccy Quote.Code Exch.Rate. CHF CHF 0 24.15 DEM BEF 2 4.85 FRF FRF 1 60.85 GBP GBP 60.85 ITL ITL 3 27.40 Means as follows - 24.15 BEF for 1 CHF. 4.86 DEM for 100 BEF. 60.85 BEF for 10 FRF. 60.85 BEF for 1 GBP. 27.40 BEF for 1,000 ITL. As the rates in fields Exch.rate and Rpt.Rate are the rate between the reporting currency and the appropriate local currency, then the base currency must be one of these two currencies. Validation Rules: The 3 digit currency code or the 3 digit numeric currency code. Multi value field associated with fields 2.1 and 4.1 to 6.1. Mandatory input. Must be a valid Currency code on the Currency.Param file. Must be either the report currency from the key or the associated local currency.

        Returns:

        public TField getExchRate()
        Returns the ExchRate

        Field description: Identifies the method of quotation of the exchange rate. This field contains the quotation code which indicates the number of units of the base currency to be quoted against. This field is used in conjunction with the field Base.Cyy (field 3), which specifies the base currency for the quotation. The rate enter in the field Exch.Rate (field 5) is the amount in the non base currency for X of the base currency. Where X depends upon the value in this field Quote.Code as follows - Value Meaning 0 or Blank 1 UNIT of the base currency 1 10 UNITS " " 2 100 UNITS " " 3 1000 UNITS " " 4 10000 UNITS " " 5 100000 UNITS " " Thus the following examples with a reporting currency of BEF. - Local.ccy Base.ccy Quote.Code Exch.Rate. CHF CHF 0 24.15 DEM BEF 2 4.85 FRF FRF 1 60.85 GBP GBP 60.85 ITL ITL 3 27.40 Means as follows - 24.15 BEF for 1 CHF. 4.86 DEM for 100 BEF. 60.85 BEF for 10 FRF. 60.85 BEF for 1 GBP. 27.40 BEF for 1,000 ITL. Validation Rules: Blank or 1 numeric character in the range 0 to 5. Multi value field associated with fields 2.1 to 3.1 and 5.1 to 6.1. (Optional input. Default value = blank.) Must be either blank or in the range 0 to 5. If Local.Ccy field is equal to Reporting currency then quotation code not allowed.

        Returns:
