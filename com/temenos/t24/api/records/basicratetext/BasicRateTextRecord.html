        public void addDescription(TField param)
        Appends the specified Description to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: Describes the Basic Interest Rate Table ID to enable it to be easily identified. This field can be expanded to allow the user to enter the Description of the Basic Interest Rate in various languages. Each expansion of the file will correspond to a new language as defined in the LANGUAGE table and will indicate to the user the language in which the Description must be entered. Validation Rules: 3-15 type A (alphanumeric) characters. Multilanguage field. Multivalue field. (Mandatory Input)

        param - the Description to append
