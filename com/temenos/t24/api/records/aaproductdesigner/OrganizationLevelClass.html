      • addOrganizationCode

        public void addOrganizationCode(TField param)
        Appends the specified OrganizationCode to the end of the collection.
        Throws an UnsupportedOperationException if the object is not modifiable.

        Field description: This field is used in connection with ORGANIZATION.LEVEL to define availability of the product in parts of the organisation. One or more codes can be specified, at least one code is mandatory for any given level. Prerequisite: An organisation structure must be setup in ST.ORGANIZATION.STRUCTURE. For example: To make a product available in BRANCH 1311 and 1312 only, select BRANCH as LEVEL and define 1311 and 1312 as the corresponding CODEs.
