      • getPreAdviceType

        public TField getPreAdviceType()
        Returns the PreAdviceType

        Field description: Used when the automatic bond redemption facility is requested - as it dictates the type of advice run to be processed. In this instance whether it's a pre advice type or not. A pre advice type transaction will remove stock from base security and place it on an identical security no. but with a suffix '-234'. This is the acknowledged early redemption indicator. Note: This type is used for the Automatic Redemption request where the System indicates that the type of Redemption should be a Final type. System cannot possibly identify early or partial draw downs - therefore it can only create a pre advised Final Redemption. Validation Rules: 1-5 Alpha numeric characters (Type S). Mandatory input. Must exist as a valid Redemption Type.
