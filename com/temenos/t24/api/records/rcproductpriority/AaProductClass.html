        public TField getAaContractPrty()
        Returns the AaContractPrty

        Field description:

        This field is used to link to the transaction level priorities that have been set up in the application RC.CONTRACT.PRIORITY and will be applied to the AA.PRODUCT.

        Forms part of a multi value set with AA.PRODUCT.

        Validation rules

        Mandatory input is the @id is an AA.PRODUCT.GROUP

        No Input field for non AA

        Must be a valid record on RC.CONTRACT.PRIORITY

        Defining a record with CUSTOM Due Rule will not follow Rank Based prioritisation
