      • getFieldType

        public TField getFieldType()
        Returns the FieldType

        Field description: Specifies if special processing is required to the FIELD.VALUE defined before comparison can take place. A value of PERIOD will result in the FIELD.VALUE being added to the current T24 system date. For example a condition may be defined for bonds maturing in 6 months. This would be defined as FIELD.VALUE of 6M. If PERIOD is defined in this field a value of today plus 6 months would be compared to the bond maturity date. The type of field which requires special processing to determine the condition. Possible values: PERIOD - Used to determine a period on from a date.

        Returns:
