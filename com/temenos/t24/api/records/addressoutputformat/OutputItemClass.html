      • getMaxLength

        public TField getMaxLength()
        Returns the MaxLength

        Field description: Indicates the maximum number of characters for an output item line. The system will evaluate the definition of the output line and will compare the result with the number of characters per line and the number of lines per output items. If the length of the result is less than max length than one line will be returned for the output item. In case the number of lines per output>1, the system will return the first chars up to max length for the first line and then will continue with the output for the next lines. If the max line is exceeded the next chars will be ignored. This is a mandatory field with the maximum length of 3.

        Parameters:
        Returns:
        TField - The MaxLength

      • getNoLinesPerOutputitem

        public TField getNoLinesPerOutputitem()
        Returns the NoLinesPerOutputitem

        Field description: Represents the maximum number of lines to use for an output item. The maximum number of lines per output item combined with the maximum number of characters per output item gives an indication of the name or address attributes to be displayed on every line part of the output. This is a mandatory field with the maximum length of 2. The default value is '1'

        Parameters:

      • getOutputItem

        public TField getOutputItem()
        Returns the OutputItem

        Field description: Identifies the number of an item in the output. An output item can have one or more lines. By giving the same name to the Output Item to different output items, the system will evaluates the first output line and if this is null it will continue with the next one. Different output item names implies that each output items/ will be included in the address output. This is a multi-value mandatory field. Validation Rules: The value for this field can be of free-text with the maximum length of 35.

        Parameters:
        Returns:
        TField - The OutputItem
