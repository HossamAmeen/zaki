      • getMinRoundAmt

        public TField getMinRoundAmt()
        Returns the MinRoundAmt

        Field description: This indicates whether the minimum amount, or the lowest unit of currency defined in the TEST.ROUNDING, has been used for rounding the input amount. The TEST.ROUNDING application either returns blank or the actual minimum unit used for the rounding. If this field is returned with a blank, then the rounding has been done to the NO.OF.DECIMAL.PLACES in the Currency record. Note: Even if in the Currency record a MIN.ROUND.AMOUNT is defined and if in the EB.ROUNDING.RULE, the USE.MIN.UNIT has not been set to ‘Y’ , then the NO.OF.DECIMAL.PLACES in the Currency record is used. In this case a blank value is returned in the field MIN.ROUND.AMT.

        Parameters:
        Returns:
        TField - The MinRoundAmt

      • getRoundingParam

        public TField getRoundingParam()
        Returns the RoundingParam

        Field description: This field indicates what parameter has been used for the rounding. The possible values are ‘ROUNDING.RULE’ and ‘CURRENCY’. If no rounding rule has been defined in the TEST.ROUNDING record, then ‘CURRENCY’ is returned. If a rounding rule has been defined in the TEST.ROUNDING record, then ‘ROUNDING.RULE’ is returned. The only exception is when the ROUNDING.TYPE in the EB.ROUNDING.RULE is set to ‘CURRENCY’ or ‘NONE’, in this case ‘CURRENCY’ is returned. Validation Rules: · It is a non-input field. The possible returned values are ‘CURRENCY’ and ‘ROUNDING.RULE’

        Parameters:
        Returns:
        TField - The RoundingParam

      • getRoundingRule

        public TField getRoundingRule()
        Returns the RoundingRule

        Field description: This defines whether a rounding rule is to be used for the rounding calculation. If it is not set then the TEST.ROUNDING uses the rounding definition at the Currency level defined in the TEST.ROUNDING record. If it is set then, the application fetches the rounding rule information defining the rounding type and any minimum unit to be used. The other required information are retrieved from the Currency record. Validation Rules: · Either left blank or a valid Id to EB.ROUNDING.RULE.

        Parameters:
        Returns:
        TField - The RoundingRule
