        public TField getInwardCarriers()
        Returns the InwardCarriers

        Field description: Specifies the valid inward carriers. Note: since one program handles inward and outward telexes, Telex is input only as an outward carrier. An INWARD CARRIER may only be changed or removed if the corresponding SHUT IN CARRIER Field is 'C' - closed.

        'SWIFT', 'EUCLID' or nothing. INWARD CARRIERS and SHUT IN CARRIER (Field 6.1) form a multivalue set listing the inward carriers and giving their status. (Optional input)

      • getShutInCarrier

        public TField getShutInCarrier()
        Returns the ShutInCarrier

        Field description: Specifies the status of Carrier Control for incoming messages from the carrier named in the previous field. In addition to 'N' and 'U' (described above) this field may also contain the following values that cannot be set on-line: 'C' Shutdown completed. 'E' End of Period processing is being run. 'A' The relevant Inward Carrier Control is active.

        'N' requests a normal shutdown after processing of all incoming messages for INWARD CARRIERS (Field 5.1) has finished. or 'U' requests an urgent shutdown after processing of the current message has been completed. INWARD CARRIERS and SHUT IN CARRIER form a multivalue set listing the inward carriers and giving their status.
