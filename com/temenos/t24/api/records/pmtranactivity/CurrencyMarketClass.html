        public TField getAsstLiabCd()
        Returns the AsstLiabCd

        Field description: Indicates the asset/liability code for each individual activity of the deal. The value of the code is assigned according to the nature of each activity. For example, for a Money Market placement: - The value date principal activity will receive the code value of "2" (liability) because it refers to an activity generating OUTgoing funds. - The maturity principal activity will receive the code value of "1" (Asset) because it refers to an activity generating Incoming funds. - The Maturity Interest activity will receive the code value of "1" (Asset) because it refers to an activity generating INcoming funds Opposite code values will be generated for a M.M.       • getRate

        public TField getRate()
        Returns the Rate

        Field description: Indicates either the exchange rate or the interest rate of the transaction. For Money Market and Loan transactions this field will contain the interest rate specified on the contract. It will only be shown on Maturity principal activity and on the Interest activity. For MM and LD transactions done on a floating rate basis, this field will contain the amount value of the rate adjusted by the margin/spread. For Forex transactions, this field will specify the exchange rate for the base currency or the inverted rate for the non base currency. For Data Caputre and Funds Transfer, this field is not used. Validation Rules: 1 to 10 characters, standard rate format. System generated field. (Multi-value associated with fields 1.1 to 9.1 and fields 11.1 to 17.1.)

        Returns:
