      • getDayNo

        public TField getDayNo()
        Returns the DayNo

        Field description: Specifies the booking date of the entries which generated the credit and debit turnover and resulting balance in the corresponding field range TOTAL.CREDITS to BALANCE. The multivalue associated group of field range DAY.NO to BALANCE contain the total amount of credit entries, the total amount of debit entries and the resulting Balance for each Booking Date for which there have been entries. The group is repeated for each different Booking Date. Entries with an ACCOUNTING.DATE and updated to Post Closing (PC) Database would be reported against the date of the earliest Post Closing (PC) Database updated (i.e. earliest PC.PERIOD.END in the entries, provided the field PC.APPLIED=Y). Validation Rules: 2 numeric characters. This is a NOINPUT field.
